## 第一部分：为什么需要逻辑回归？

### 🏥 医学诊断例子

假设我们要判断一个病人是否患有某种疾病：

```
特征：
- 血压：x₁
- 血糖：x₂
- 胆固醇：x₃

目标是：判断有病（1）还是没病（0）
```

**线性回归能解决这个问题吗？**

```
线性回归输出：y = 0.7（预测值可以是任意实数）
但我们需要的是：患病概率 P(有病)，范围在 [0, 1]
```

线性回归的问题：
- 输出可以是任意实数（-∞ 到 +∞）
- 无法直接解释为概率

**解决方案：逻辑回归！**

## 第二部分：Sigmoid 函数

### 核心思想

```
线性部分：z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b（可以是任意实数）
激活函数：σ(z) = 1 / (1 + e^(-z))（输出限制在 0-1 之间）
```

### Sigmoid 的数学形式

```
σ(z) = 1 / (1 + e^(-z))

其中 e ≈ 2.71828（自然常数）
```

### Sigmoid 的性质

```python
def sigmoid(z):
    return 1 / (1 + np.exp(-z))

print(f"sigmoid(0)   = {sigmoid(0):.4f}")    # 0.5000（50%概率）
print(f"sigmoid(2)   = {sigmoid(2):.4f}")    # 0.8808（88%概率）
print(f"sigmoid(-2)  = {sigmoid(-2):.4f}")   # 0.1192（12%概率）
print(f"sigmoid(10)  = {sigmoid(10):.4f}")   # 0.99995
print(f"sigmoid(-10) = {sigmoid(-10):.4f}")  # 0.00005
```

### Sigmoid 的直观理解

```
σ(z)
1 │───────────────
  │            ╱
  │          ╱
  │        ╱
  │      ╱
  │    ╱
  │  ╱
0.5│/
  └──────────────── z
   -5    0     5

当 z > 0，输出 > 0.5（倾向正类）
当 z < 0，输出 < 0.5（倾向负类）
当 z = 0，输出 = 0.5（不确定）
```

## 第三部分：决策边界

### 什么是决策边界？

**决策边界 = 使得 P(y=1) = 0.5 的点组成的线**

```
因为 P(y=1) = σ(z) = 0.5 时，解得 z = 0

所以决策边界就是：w₁x₁ + w₂x₂ + ... + wₙxₙ + b = 0
```

## 第四部分：损失函数

### 为什么不能用 MSE？

```python
# 如果真实是 1，预测是 0.1
loss = (1 - 0.1)² = 0.81  # 看起来还行
# 但实际上这个预测非常自信地错了！
```

### 更好的选择：交叉熵损失（Log Loss）

```
当 y = 1 时：L = -log(ŷ)
当 y = 0 时：L = -log(1 - ŷ)

合并成一个公式：
L = -[y × log(ŷ) + (1-y) × log(1-ŷ)]
```

```python
def cross_entropy_loss(y_true, y_pred):
    eps = 1e-15
    y_pred = np.clip(y_pred, eps, 1 - eps)
    loss = -(y_true * np.log(y_pred) + (1-y_true) * np.log(1-y_pred))
    return np.mean(loss)
```

## 第五部分：梯度下降推导

### 推导过程

```
∂L/∂w = (ŷ - y) × x
∂L/∂b = (ŷ - y)
```

### 代码实现

```python
def logistic_regression_gd(X, y, lr=0.1, epochs=1000):
    n_samples, n_features = X.shape
    w = np.zeros(n_features)
    b = 0.0

    for epoch in range(epochs):
        z = np.dot(X, w) + b
        y_pred = sigmoid(z)
        dz = y_pred - y

        dw = (1/n_samples) * np.dot(X.T, dz)
        db = (1/n_samples) * np.sum(dz)

        w -= lr * dw
        b -= lr * db

    return w, b
```

## ✅ 小结

1. **逻辑回归**名字有"回归"，实际是分类
2. 核心：**Sigmoid 函数**把线性输出转为概率
3. 决策边界：w₁x₁ + ... + b = 0
4. 损失函数：**交叉熵损失**
