# 逻辑回归

_二分类问题的经典方法_

---

## 第一部分：什么是逻辑回归？

### 核心思想

**虽然名字叫"回归"，但它是用来做分类的！**

```
线性回归：y = wx + b  → 输出连续值
逻辑回归：p = sigmoid(wx + b)  → 输出概率 [0, 1]
```

### Sigmoid 函数

```python
def sigmoid(z):
    return 1 / (1 + np.exp(-z))
```

---

## 第二部分：代码实现

```python
def logistic_regression(X, y, lr=0.01, epochs=1000):
    m, n = X.shape
    w = np.zeros(n)
    b = 0

    for epoch in range(epochs):
        z = np.dot(X, w) + b
        p = sigmoid(z)

        dw = (1 / m) * np.dot(X.T, (p - y))
        db = (1 / m) * np.sum(p - y)

        w -= lr * dw
        b -= lr * db

    return w, b
```
