# 线性回归

_预测连续值的经典方法_

---

## 第一部分：什么是线性回归？

### 核心思想

**用一条直线拟合数据点，使得预测值与真实值的误差最小。**

```
y = wx + b

w: 斜率（权重）
b: 截距（偏置）
```

---

## 第二部分：代码实现

### 最小二乘法

```python
import numpy as np

def linear_regression(x, y):
    n = len(x)
    x = np.array(x)
    y = np.array(y)

    x_mean = np.mean(x)
    y_mean = np.mean(y)

    numerator = np.sum((x - x_mean) * (y - y_mean))
    denominator = np.sum((x - x_mean) ** 2)

    w = numerator / denominator
    b = y_mean - w * x_mean

    return w, b

# 测试
x = [1, 2, 3, 4, 5]
y = [2.1, 4.0, 5.9, 8.1, 10.2]
w, b = linear_regression(x, y)
print(f"y = {w:.2f}x + {b:.2f}")
```

### 梯度下降法

```python
def gradient_descent(x, y, lr=0.01, epochs=1000):
    m = len(x)
    w = 0.0
    b = 0.0

    for epoch in range(epochs):
        y_pred = w * x + b
        error = y_pred - y

        w -= lr * (2 / m) * np.sum(error * x)
        b -= lr * (2 / m) * np.sum(error)

    return w, b
```
