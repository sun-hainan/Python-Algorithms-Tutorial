## 第一部分：什么是线性回归？

### 🏠 生活例子：预测房价

假设你是一个房产中介，客户问你：
> "一套 120 平米的房子，大概能卖多少钱？"

你会怎么回答？

你可能会想：
- 面积越大，价格越高
- 如果我知道 100 平米的房子卖 300 万
- 120 平米的房子大概卖... 360 万？

这就是**线性回归**的思想！

```
房价 ≈ 单价 × 面积 + 基础价格

比如：房价 = 3万/m² × 面积(m²) + 50万（基础价格）
```

---

## 第二部分：数学形式

### 一元线性回归（一个特征）

```
y = wx + b

其中：
y 是我们要预测的值（房价）
x 是已知的特征（面积）
w 是斜率（每平米多少钱）← 要学习的参数
b 是截距（基础价格）  ← 要学习的参数
```

### 多元线性回归（多个特征）

```
y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

比如预测房价时：
x₁ = 面积
x₂ = 房间数
x₃ = 距地铁距离
x₄ = 房龄

y = 3.5万×面积 + 10万×房间数 - 5万×距地铁 + ...
```

为方便表示，我们把 b 也写成 w 的形式：

```
y = w₀ + w₁x₁ + w₂x₂ + ... + wₙxₙ

可以写成向量形式：

     ┌ w₀ ┐
W =  │ w₁ │   (n+1 维向量)
     │ ... │
     └ wₙ ┘

     ┌ 1  ┐
X =  │ x₁ │   (n+1 维向量，加入 x₀ = 1)
     │ ... │
     └ xₙ ┘

则 y = W · X = W^T X
```

## 第三部分：如何找到最好的 w 和 b？

### 损失函数：衡量"多差"

当我们用 `y = wx + b` 做预测时，每次预测都会有误差：

```
真实值：y_actual = 360万
预测值：y_pred = wx + b = 3.5 × 120 + 20 = 440万

误差 = y_actual - y_pred = 360 - 440 = -80万（我们多估了）
```

我们希望误差越小越好。但是误差有正有负，直接相加会抵消。

**解决方案：平方误差！**

```
损失函数 L(w, b) = (y_actual - y_pred)²

用平方而不是绝对值的原因：
1. 平方永远是正数
2. 越大误差，惩罚越重
3. 数学上求导方便
```

对于多个样本：

```
L(w, b) = Σᵢ (y_i - (wx_i + b))²

展开写：
L = (y₁ - wx₁ - b)² + (y₂ - wx₂ - b)² + ... + (yₙ - wxₙ - b)²
```

**我们的目标：找到 w 和 b，使得 L 最小！**

---

## 第四部分：方法一：最小二乘法

### 推导过程

对于一元线性回归：`y = wx + b`

我们要最小化：`L = Σ(y_i - wx_i - b)²`

**对 b 求导并设为 0：**

```
∂L/∂b = Σ 2(y_i - wx_i - b)(-1) = 0

化简：Σ(y_i - wx_i - b) = 0
     Σy_i - wΣx_i - nb = 0

     b = (Σy_i - wΣx_i) / n
       = ȳ - w x̄

其中：
ȳ = Σy_i / n（y 的平均值）
x̄ = Σx_i / n（x 的平均值）
```

**对 w 求导并设为 0：**

```
∂L/∂w = Σ 2(y_i - wx_i - b)(-x_i) = 0

化简：Σ x_i(y_i - wx_i - b) = 0
     Σ x_i y_i - wΣ x_i² - bΣ x_i = 0

代入 b = ȳ - w x̄：

Σ x_i y_i - wΣ x_i² - (ȳ - w x̄)Σ x_i = 0
Σ x_i y_i - wΣ x_i² - ȳΣ x_i + w x̄ Σ x_i = 0
Σ x_i y_i - ȳΣ x_i - w(Σ x_i² - x̄ Σ x_i) = 0

所以：
w = (Σ x_i y_i - ȳΣ x_i) / (Σ x_i² - x̄ Σ x_i)

写成更常见的形式：
w = Σ(x_i - x̄)(y_i - ȳ) / Σ(x_i - x̄)²
```

这就是**最小二乘法**的闭式解！

---

### 代码实现

```python
import numpy as np

def linear_regression_least_squares(x, y):
    """
    最小二乘法求解线性回归

    参数:
        x: 自变量数组 (n,)
        y: 因变量数组 (n,)
    返回:
        w: 斜率
        b: 截距
    """
    # 确保是 numpy 数组
    x = np.array(x)
    y = np.array(y)

    n = len(x)

    # 计算平均值
    x_mean = np.mean(x)
    y_mean = np.mean(y)

    # ===== 详细推导中的公式 =====
    # w = Σ(x_i - x̄)(y_i - ȳ) / Σ(x_i - x̄)²
    numerator = np.sum((x - x_mean) * (y - y_mean))  # Σ(x_i - x̄)(y_i - ȳ)
    denominator = np.sum((x - x_mean) ** 2)            # Σ(x_i - x̄)²

    w = numerator / denominator
    b = y_mean - w * x_mean

    return w, b


# ========== 测试 ==========
# 假设我们有 5 个数据点：面积 -> 房价
areas = np.array([80, 100, 120, 140, 160])  # 面积（平米）
prices = np.array([280, 340, 400, 450, 520])  # 房价（万）

w, b = linear_regression_least_squares(areas, prices)

print(f"拟合结果：房价 = {w:.2f} × 面积 + {b:.2f}")
# 房价 = 2.93 × 面积 + 45.33

# 预测：130 平米的房子
area_new = 130
price_pred = w * area_new + b
print(f"130平米的房子预测价格：{price_pred:.2f} 万")
# 约 426 万
```

## 第五部分：方法二：梯度下降法

### 直观理解

想象你在一座山上，想要找到山谷最低点（全局最低）。

```
        山顶
       /    \
      /      \
     /   ★    \    ← 你在山顶
    /          \
   /      ↓↓↓     \   ← 沿着最陡的方向下山
 /                \
/                  \
===================== 山谷（最低点）
```

梯度下降的步骤：
1. 看看四周，找到最陡的下坡方向
2. 迈一小步
3. 重复 1-2 直到几乎不再下降

**梯度 = 函数在某一点上升最快的方向**
**所以负梯度 = 下降最快的方向**

---

### 推导过程

对于线性回归：`y_pred = wx + b`
损失函数：`L = Σ(y_i - y_pred_i)² / n`

**梯度 = 损失函数对参数的偏导数**

**1. 对 w 求偏导：**

```
L = Σ(y_i - wx_i - b)² / n

∂L/∂w = (1/n) × Σ 2(y_i - wx_i - b) × ∂(y_i - wx_i - b)/∂w
       = (2/n) × Σ (y_i - wx_i - b) × (-x_i)
       = -(2/n) × Σ x_i(y_i - wx_i - b)
```

**2. 对 b 求偏导：**

```
∂L/∂b = (2/n) × Σ (y_i - wx_i - b) × (-1)
       = -(2/n) × Σ (y_i - wx_i - b)
```

**所以梯度下降的更新规则是：**

```
w_new = w_old - learning_rate × ∂L/∂w
       = w_old + learning_rate × (2/n) × Σ x_i(y_i - wx_i - b)

b_new = b_old - learning_rate × ∂L/∂b
       = b_old + learning_rate × (2/n) × Σ (y_i - wx_i - b)
```

---

### 代码实现

```python
def gradient_descent(x, y, learning_rate=0.01, epochs=1000):
    """
    梯度下降法求解线性回归

    参数:
        x: 自变量数组 (n,)
        y: 因变量数组 (n,)
        learning_rate: 学习率（步长），控制每一步走多远
        epochs: 迭代次数
    返回:
        w, b: 拟合的参数
    """
    x = np.array(x)
    y = np.array(y)
    n = len(x)

    # ========== 初始化 ==========
    # 随机初始化 w 和 b（也可以初始化为 0）
    w = 0.0
    b = 0.0

    # ========== 迭代优化 ==========
    for epoch in range(epochs):

        # ----- 计算预测值 -----
        # y_pred = w * x + b，对每个样本计算预测值
        y_pred = w * x + b

        # ----- 计算误差 -----
        # 误差 = 真实值 - 预测值
        error = y - y_pred  # 这是一个向量，包含每个样本的误差

        # ----- 计算梯度 -----
        # ∂L/∂w = -(2/n) × Σ x_i × error
        grad_w = -(2/n) * np.sum(error * x)

        # ∂L/∂b = -(2/n) × Σ error
        grad_b = -(2/n) * np.sum(error)

        # ----- 更新参数 -----
        # 负梯度方向走，learning_rate 控制步长
        w = w - learning_rate * grad_w
        b = b - learning_rate * grad_b

        # ----- 打印进度（每 100 轮）-----
        if epoch % 100 == 0:
            # 计算当前损失值
            loss = np.mean(error ** 2)
            print(f"Epoch {epoch:4d}: w = {w:.4f}, b = {b:.4f}, loss = {loss:.4f}")

    return w, b


# ========== 测试 ==========
areas = np.array([80, 100, 120, 140, 160])
prices = np.array([280, 340, 400, 450, 520])

print("=== 梯度下降过程 ===")
w, b = gradient_descent(areas, prices, learning_rate=0.0001, epochs=1000)

print(f"\n最终结果：房价 = {w:.4f} × 面积 + {b:.4f}")
```

---

### 学习率的影响

```
学习率太大：步子迈得太大，会在最低点附近"振荡"，甚至越过最低点发散

学习率太小：步子太小，收敛太慢，需要很多轮迭代
```

## 第六部分：评估指标

### 1. MSE（均方误差）

```python
def mean_squared_error(y_true, y_pred):
    """
    MSE = 平均值(真实值 - 预测值)²

    物理意义：误差的平方的平均值
    缺点：对离群点（很大误差）惩罚很重
    """
    return np.mean((y_true - y_pred) ** 2)


# 例子
y_true = np.array([280, 340, 400, 450, 520])
y_pred = np.array([278, 342, 398, 452, 518])
mse = mean_squared_error(y_true, y_pred)
print(f"MSE = {mse:.2f}")  # MSE = 6.8
```

### 2. RMSE（均方根误差）

```python
def root_mean_squared_error(y_true, y_pred):
    """
    RMSE = √MSE

    物理意义：和原始数据同单位，方便理解
    """
    return np.sqrt(mean_squared_error(y_true, y_pred))

# 例子
rmse = root_mean_squared_error(y_true, y_pred)
print(f"RMSE = {rmse:.2f}")  # RMSE = 2.61
# 意味着平均预测误差约 2.6 万
```

### 3. MAE（平均绝对误差）

```python
def mean_absolute_error(y_true, y_pred):
    """
    MAE = 平均值(|真实值 - 预测值|)

    优点：对离群点不那么敏感
    """
    return np.mean(np.abs(y_true - y_pred))
```

### 4. R²（决定系数）

```python
def r2_score(y_true, y_pred):
    """
    R² = 1 - SS_res / SS_tot

    其中：
    SS_res = Σ(真实值 - 预测值)²  （残差平方和）
    SS_tot = Σ(真实值 - 平均值)²    （总平方和）

    R² 的物理意义：
    - R² = 1：完美预测
    - R² = 0：和用平均值预测一样
    - R² < 0：比平均值预测还差

    R² 越接近 1，说明模型解释能力越强
    """
    ss_res = np.sum((y_true - y_pred) ** 2)           # 残差平方和
    ss_tot = np.sum((y_true - np.mean(y_true)) ** 2)  # 总平方和
    return 1 - ss_res / ss_tot
```

## 第七部分：sklearn 实现

```python
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, mean_absolute_error, r2_score
import numpy as np

# ===== 准备数据 =====
X = np.array([[80], [100], [120], [140], [160]])  # 形状：(5, 1)
y = np.array([280, 340, 400, 450, 520])            # 形状：(5,)

# ===== 创建模型并训练 =====
model = LinearRegression()        # 创建线性回归模型
model.fit(X, y)                   # 训练：自动用最小二乘法

# ===== 查看结果 =====
print(f"斜率 w = {model.coef_[0]:.4f}")   # 2.9286
print(f"截距 b = {model.intercept_:.4f}")  # 45.3333

# ===== 预测 =====
y_pred = model.predict(X)  # 预测训练数据

# ===== 评估 =====
print(f"MSE  = {mean_squared_error(y, y_pred):.4f}")
print(f"RMSE = {np.sqrt(mean_squared_error(y, y_pred)):.4f}")
print(f"MAE  = {mean_absolute_error(y, y_pred):.4f}")
print(f"R²   = {r2_score(y, y_pred):.4f}")

# ===== 预测新数据 =====
new_area = np.array([[130]])  # 130 平米的房子
print(f"130平米预测价格：{model.predict(new_area)[0]:.2f} 万")
```

## 第九部分：名词解释

### 什么是"线性"？

**线性 ≠ 直线**

线性回归的"线性"指的是**对参数的线性**，而不是对特征的线性！

```python
# 线性（对参数 w 是线性的）：
y = w₁x₁ + w₂x₂ + b          ✓
y = w₁x₁ + w₂x₁² + b         ✓
y = w₁sin(x₁) + w₂log(x₂) + b  ✓

# 非线性（对参数不是线性的）：
y = w₁^2 * x₁ + b              ✗
y = exp(w₁ * x₁) + b           ✗
```

## ✅ 小结

1. **线性回归**是最基础的监督学习算法
2. 模型形式：`y = wx + b`
3. **最小二乘法**：直接求解析解
   - `w = Σ(x_i - x̄)(y_i - ȳ) / Σ(x_i - x̄)²`
   - `b = ȳ - w x̄`
4. **梯度下降法**：迭代优化
   - `w = w - lr × ∂L/∂w`
5. 评估指标：MSE、RMSE、MAE、R²
