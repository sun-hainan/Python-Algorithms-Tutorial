## 第一部分：什么是感知机？

### 🎯 核心思想

**感知机是最简单的神经网络模型：输入 → 加权求和 → 激活输出。**

```
感知机结构：

输入 x1 ──w1──╲
              ╲
输入 x2 ──w2──╲╲
              ╲╲
输入 x3 ──w3──╲╲╲
              ╲╲╲
              bias ──b
               ╲╲╲
                ╲╲╲
                 ▼
              activation
                 │
                 ▼
              output y
```

### 数学表达

```
z = w1*x1 + w2*x2 + w3*x3 + b
y = activation(z)

二分类：y = 1 if z > 0 else 0
```

## 第二部分：代码实现

```python
import numpy as np

class Perceptron:
    """感知机"""
    def __init__(self, n_features):
        self.weights = np.random.randn(n_features) * 0.01
        self.bias = 0

    def forward(self, x):
        """前向传播"""
        z = np.dot(x, self.weights) + self.bias
        return np.where(z > 0, 1, 0)  # 阶跃函数

    def train(self, X, y, lr=0.1, epochs=100):
        """训练"""
        for epoch in range(epochs):
            for x_i, y_i in zip(X, y):
                y_pred = self.forward(x_i)
                error = y_i - y_pred

                # 更新权重
                self.weights += lr * error * x_i
                self.bias += lr * error

            if epoch % 20 == 0:
                preds = self.forward(X)
                acc = np.mean(preds == y)
                print(f"Epoch {epoch}: accuracy = {acc:.4f}")


# 测试：AND 逻辑门
X_and = np.array([
    [0, 0],
    [0, 1],
    [1, 0],
    [1, 1]
])
y_and = np.array([0, 0, 0, 1])

p = Perceptron(n_features=2)
p.train(X_and, y_and, epochs=100)

print("\n测试 AND 门：")
for x in X_and:
    print(f"{x} -> {p.forward(x)}")
```

## 第三部分：逻辑门实现

### AND、OR、XOR 对比

```python
# AND 门（线性可分）
"""
0 AND 0 = 0
1 AND 1 = 1  ✓ 感知机可以学习
"""

# OR 门（线性可分）
"""
0 OR 0 = 0
1 OR 1 = 1  ✓ 感知机可以学习
"""

# XOR 门（非线性可分）
"""
0 XOR 0 = 0
1 XOR 1 = 0
0 XOR 1 = 1
1 XOR 0 = 1  ✗ 感知机无法学习！
"""
```

### XOR 问题说明

```
XOR 的数据分布：

y
^
1 │  ●
  │    ╲
0 │────●─→ x
  │ 0   1

无法用一条直线分开！

解决：多层感知机（MLP）
```

## 第四部分：多层感知机 (MLP)

```python
class MLP:
    """多层感知机"""
    def __init__(self, layers):
        """
        layers: 每层的神经元数量，如 [2, 4, 1]
        """
        self.weights = []
        self.biases = []

        for i in range(len(layers) - 1):
            w = np.random.randn(layers[i], layers[i+1]) * 0.1
            b = np.zeros(layers[i+1])
            self.weights.append(w)
            self.biases.append(b)

    def sigmoid(self, z):
        return 1 / (1 + np.exp(-z))

    def forward(self, x):
        """前向传播"""
        self.activations = [x]
        for w, b in zip(self.weights, self.biases):
            z = np.dot(self.activations[-1], w) + b
            a = self.sigmoid(z)
            self.activations.append(a)
        return self.activations[-1]

    def predict(self, X):
        return (self.forward(X) > 0.5).astype(int)
```

## 第六部分：名词解释

### 激活函数

**定义：** 给神经网络引入非线性。

| 函数 | 公式 | 特点 |
|------|------|------|
| 阶跃函数 | y=1 if z>0 else 0 | 感知机用 |
| Sigmoid | 1/(1+e^(-z)) | 平滑、概率输出 |
| Tanh | (e^z - e^(-z))/(e^z + e^(-z)) | 零中心 |
| ReLU | max(0, z) | 深度学习常用 |

### 线性可分

**定义：** 可以用一条直线（或超平面）把两类点完全分开。

## ✅ 小结

1. **感知机**是最简单的神经网络模型
2. 由输入、权重、偏置、激活函数组成
3. 可以实现 AND、OR 等线性可分问题
4. **无法解决 XOR 问题**（需要多层）
5. 多层感知机 (MLP) 可以解决非线性问题
