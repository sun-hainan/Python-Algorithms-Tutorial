## 第一部分：什么是逻辑回归？

### 🎯 核心思想

**虽然名字叫"回归"，但它是用来做分类的！**

```
线性回归：y = wx + b  → 输出连续值
逻辑回归：p = sigmoid(wx + b)  → 输出概率 [0, 1]
```

### Sigmoid 函数

```python
import numpy as np

def sigmoid(z):
    """Sigmoid 激活函数"""
    return 1 / (1 + np.exp(-z))


# 测试
z = np.linspace(-10, 10, 100)
print(sigmoid(0))    # 0.5
print(sigmoid(10))  # 0.9999
print(sigmoid(-10)) # 0.00005
```

## 第三部分：代码实现

### 逻辑回归梯度下降

```python
def logistic_regression(X, y, lr=0.01, epochs=1000):
    """
    逻辑回归实现

    X: 特征矩阵 (n_samples, n_features)
    y: 标签 (n_samples,)
    """
    m, n = X.shape
    w = np.zeros(n)
    b = 0

    for epoch in range(epochs):
        # 前向传播
        z = np.dot(X, w) + b
        p = sigmoid(z)

        # 计算梯度
        dz = p - y
        dw = (1 / m) * np.dot(X.T, dz)
        db = (1 / m) * np.sum(dz)

        # 更新参数
        w -= lr * dw
        b -= lr * db

        if epoch % 100 == 0:
            # 计算损失
            loss = -np.mean(y * np.log(p + 1e-8) + (1 - y) * np.log(1 - p + 1e-8))
            print(f"Epoch {epoch}: loss = {loss:.4f}")

    return w, b


def predict(X, w, b, threshold=0.5):
    """预测"""
    p = sigmoid(np.dot(X, w) + b)
    return (p >= threshold).astype(int)
```

## 第五部分：多分类扩展

### One-vs-Rest (OvR)

```python
def one_vs_rest(X, y, lr=0.01, epochs=1000):
    """一对多策略"""
    classes = np.unique(y)
    classifiers = {}

    for cls in classes:
        # 二分类：当前类 vs 其他类
        y_binary = (y == cls).astype(int)
        w, b = logistic_regression(X, y_binary, lr, epochs)
        classifiers[cls] = (w, b)

    return classifiers


def predict_multi(X, classifiers):
    """多分类预测"""
    predictions = {}
    for cls, (w, b) in classifiers.items():
        p = sigmoid(np.dot(X, w) + b)
        predictions[cls] = p
    return max(predictions, key=predictions.get)
```

y_pred = model.predict(X_test)
print(f"准确率: {accuracy_score(y_test, y_pred):.4f}")

## 第七部分：名词解释

### 对数损失 (Log Loss)

**定义：** 衡量分类器概率预测的损失。

```python
def log_loss(y_true, y_pred):
    return -np.mean(y_true * np.log(y_pred + 1e-8) + (1 - y_true) * np.log(1 - y_pred + 1e-8))
```

### Sigmoid

**公式：** σ(z) = 1 / (1 + e^(-z))

**特点：** 将任意实数映射到 (0, 1) 区间。

## ✅ 小结

1. **逻辑回归**本质是分类，但用了"回归"的名字
2. 核心：**Sigmoid 函数**把线性输出转为概率
3. 使用交叉熵损失
4. 可以通过 OvR 扩展到多分类
5. 优点：简单、可解释、输出概率
