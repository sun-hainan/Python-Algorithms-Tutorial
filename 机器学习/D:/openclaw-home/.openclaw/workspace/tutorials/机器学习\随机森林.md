## 第一部分：什么是随机森林？

### 🎯 核心思想

**让很多棵决策树一起投票决定结果。**

```
随机森林
    │
    ├── 决策树 1 ──→ 预测: A
    ├── 决策树 2 ──→ 预测: B
    ├── 决策树 3 ──→ 预测: A
    ├── ...
    └── 决策树 N ──→ 预测: A

最终结果: A (多数投票)
```

## 第二部分：两个核心技巧

### 技巧1：Bagging（自助聚合）

```python
def bootstrap_sample(X, y, n_samples=None):
    """有放回抽样"""
    if n_samples is None:
        n_samples = len(X)

    indices = np.random.randint(0, n_samples, size=n_samples)
    return X[indices], y[indices]
```

## 第三部分：代码实现

```python
class RandomForest:
    def __init__(self, n_trees=100, max_depth=10, min_samples_split=2, n_features_subset=None):
        self.n_trees = n_trees
        self.max_depth = max_depth
        self.min_samples_split = min_samples_split
        self.n_features_subset = n_features_subset
        self.trees = []

    def fit(self, X, y):
        self.trees = []
        for _ in range(self.n_trees):
            # Bagging
            X_boot, y_boot = bootstrap_sample(X, y)

            # 训练决策树
            tree = DecisionTree(
                max_depth=self.max_depth,
                min_samples_split=self.min_samples_split,
                n_features_subset=self.n_features_subset
            )
            tree.fit(X_boot, y_boot)
            self.trees.append(tree)

    def predict(self, X):
        # 多数投票
        predictions = np.array([tree.predict(X) for tree in self.trees])
        # 每行是一棵树的结果，取众数
        return np.apply_along_axis(lambda x: np.bincount(x).argmax(), axis=0, arr=predictions)

    def predict_proba(self, X):
        """预测概率"""
        proba = np.mean([tree.predict(X) for tree in self.trees], axis=0)
        return proba
```

## 第四部分：实际应用场景

### 🏦 场景1：信用评估

```python
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 生成数据
X, y = make_classification(n_samples=1000, n_features=20, n_classes=2)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# 训练随机森林
rf = RandomForest(n_trees=100, max_depth=10)
rf.fit(X_train, y_train)

# 预测
y_pred = rf.predict(X_test)
print(f"准确率: {accuracy_score(y_test, y_pred):.4f}")
```

### 📊 场景2：特征重要性

```python
def feature_importance(rf, feature_names):
    """计算特征重要性"""
    importances = np.zeros(len(feature_names))
    for tree in rf.trees:
        importances += tree.feature_importances_
    importances /= rf.n_trees

    # 排序
    indices = np.argsort(importances)[::-1]
    for i in indices[:10]:
        print(f"{feature_names[i]}: {importances[i]:.4f}")
```

## 第六部分：随机森林 vs 单棵决策树

| 对比 | 单棵决策树 | 随机森林 |
|------|------------|----------|
| 稳定性 | 容易过拟合 | 稳定，不易过拟合 |
| 方差 | 高 | 低（平均减少方差）|
| 可解释性 | 强 | 弱（多棵树）|
| 训练速度 | 快 | 慢 |
| 准确率 | 较低 | 较高 |

---

## 第七部分：名词解释

### Bagging

**定义：** Bootstrap Aggregating，自助聚合。训练多个模型，每个模型在不同的自助样本上训练，最后投票。

### Out-of-Bag (OOB)

**定义：** 对于每棵树，大约有 1/3 的样本没有被抽中，可以用来做验证。

---

## ✅ 小结

1. **随机森林**是决策树的集成
2. 两个核心技巧：**Bagging** + **特征子采样**
3. 通过多数投票减少方差，提高稳定性
4. 优点：不易过拟合、可处理高维数据
5. 缺点：可解释性差、训练慢
