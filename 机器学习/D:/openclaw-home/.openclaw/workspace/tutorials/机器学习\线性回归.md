## 第一部分：什么是线性回归？

### 🎯 核心思想

**用一条直线拟合数据点，使得预测值与真实值的误差最小。**

```
y = wx + b

w: 斜率（权重）
b: 截距（偏置）
```

---

## 第二部分：图解过程

```
拟合直线：
y
^
│    * (数据点)
│   *
│  *
│ *
│*─────────────
0           x

y = 0.5x + 1.2 (拟合的直线)
```

## 第三部分：代码实现

### 最小二乘法

```python
import numpy as np

def linear_regression(x, y):
    """
    线性回归：y = wx + b

    使用最小二乘法求解
    """
    n = len(x)
    x = np.array(x)
    y = np.array(y)

    # 计算 w 和 b
    x_mean = np.mean(x)
    y_mean = np.mean(y)

    numerator = np.sum((x - x_mean) * (y - y_mean))
    denominator = np.sum((x - x_mean) ** 2)

    w = numerator / denominator
    b = y_mean - w * x_mean

    return w, b


# 测试
x = [1, 2, 3, 4, 5]
y = [2.1, 4.0, 5.9, 8.1, 10.2]
w, b = linear_regression(x, y)
print(f"y = {w:.2f}x + {b:.2f}")
# y = 2.02x + 0.04
```

### 梯度下降法

```python
def gradient_descent(x, y, lr=0.01, epochs=1000):
    """
    梯度下降法求解线性回归
    """
    m = len(x)
    x = np.array(x)
    y = np.array(y)

    w = 0.0
    b = 0.0

    for epoch in range(epochs):
        y_pred = w * x + b
        error = y_pred - y

        # 计算梯度
        w_grad = (2 / m) * np.sum(error * x)
        b_grad = (2 / m) * np.sum(error)

        # 更新参数
        w -= lr * w_grad
        b -= lr * b_grad

        if epoch % 100 == 0:
            loss = np.mean(error ** 2)
            print(f"Epoch {epoch}: loss = {loss:.4f}")

    return w, b


w, b = gradient_descent(x, y)
print(f"y = {w:.2f}x + {b:.2f}")
```

## 第四部分：评估指标

| 指标 | 说明 | 公式 |
|------|------|------|
| MSE | 均方误差 | mean((y - y_pred)²) |
| RMSE | 均方根误差 | sqrt(MSE) |
| MAE | 平均绝对误差 | mean(|y - y_pred|) |
| R² | 决定系数 | 1 - SS_res / SS_tot |

```python
def evaluate(y_true, y_pred):
    """评估模型"""
    mse = np.mean((y_true - y_pred) ** 2)
    rmse = np.sqrt(mse)
    mae = np.mean(np.abs(y_true - y_pred))

    ss_res = np.sum((y_true - y_pred) ** 2)
    ss_tot = np.sum((y_true - np.mean(y_true)) ** 2)
    r2 = 1 - ss_res / ss_tot

    print(f"MSE:  {mse:.4f}")
    print(f"RMSE: {rmse:.4f}")
    print(f"MAE:  {mae:.4f}")
    print(f"R²:   {r2:.4f}")

    return mse, rmse, mae, r2
```

## 第七部分：名词解释

### 最小二乘法

**定义：** 找到一条直线，使得所有数据点到直线的垂直距离平方和最小。

### 梯度下降

**定义：** 通过不断沿着梯度方向更新参数，逐步逼近最优解。

---

## ✅ 小结

1. **线性回归**用直线拟合数据：y = wx + b
2. 两种求解方法：最小二乘法、梯度下降
3. 评估指标：MSE、RMSE、MAE、R²
4. 优点：简单、可解释性强
5. 缺点：只能处理线性关系
