## 第一部分：什么是梯度下降？

### 🎯 核心思想

**沿着梯度的方向，逐步找到函数的最小值。**

```
想象你站在山坡上，想走到山谷最低点：
1. 看看哪个方向是下坡
2. 迈一步
3. 重复直到到达最低点

梯度 = 告诉你在哪个方向下坡
学习率 = 决定你迈多大步
```

---

## 第二部分：图解过程

```
损失函数曲面：
    /\
   /  \
  /    \      ← 局部最小
 /      \   /
/        \ /
          \    ← 全局最小
           \

梯度下降：
从山顶开始，沿着最陡的下坡方向
一步步走到山谷
```

## 第三部分：代码实现

### 批量梯度下降 (BGD)

```python
import numpy as np

def gradient_descent(X, y, lr=0.01, epochs=1000):
    """
    批量梯度下降

    每次使用全部数据计算梯度
    """
    m, n = X.shape
    w = np.zeros(n)
    b = 0

    for epoch in range(epochs):
        # 预测
        y_pred = np.dot(X, w) + b

        # 计算梯度
        dw = (1 / m) * np.dot(X.T, (y_pred - y))
        db = (1 / m) * np.sum(y_pred - y)

        # 更新
        w -= lr * dw
        b -= lr * db

        if epoch % 100 == 0:
            loss = (1 / (2 * m)) * np.sum((y_pred - y) ** 2)
            print(f"Epoch {epoch}: loss = {loss:.4f}")

    return w, b
```

### 随机梯度下降 (SGD)

```python
def sgd(X, y, lr=0.01, epochs=1000, batch_size=32):
    """
    随机梯度下降

    每次使用一个或少量样本计算梯度
    """
    m = len(y)

    for epoch in range(epochs):
        # 打乱数据
        indices = np.random.permutation(m)

        for i in range(0, m, batch_size):
            batch_idx = indices[i:i+batch_size]
            X_batch = X[batch_idx]
            y_batch = y[batch_idx]

            # 预测
            y_pred = np.dot(X_batch, w) + b

            # 计算梯度
            dw = (1 / len(batch_idx)) * np.dot(X_batch.T, (y_pred - y_batch))
            db = (1 / len(batch_idx)) * np.sum(y_pred - y_batch)

            # 更新
            w -= lr * dw
            b -= lr * db

        if epoch % 100 == 0:
            loss = (1 / m) * np.sum((np.dot(X, w) + b - y) ** 2)
            print(f"Epoch {epoch}: loss = {loss:.4f}")

    return w, b
```

### 小批量梯度下降 (Mini-Batch GD)

```python
def mini_batch_gd(X, y, lr=0.01, epochs=1000, batch_size=32):
    """
    小批量梯度下降

    每次使用 batch_size 个样本
    """
    m = len(y)

    for epoch in range(epochs):
        indices = np.random.permutation(m)

        for i in range(0, m, batch_size):
            batch_idx = indices[i:i+batch_size]
            X_batch = X[batch_idx]
            y_batch = y[batch_idx]

            y_pred = np.dot(X_batch, w) + b
            dw = (1 / len(batch_idx)) * np.dot(X_batch.T, (y_pred - y_batch))
            db = (1 / len(batch_idx)) * np.sum(y_pred - y_batch)

            w -= lr * dw
            b -= lr * db

    return w, b
```

## 第四部分：三种方法对比

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| BGD | 稳定收敛 | 慢、大数据难处理 | 小数据集 |
| SGD | 快、在线学习 | 震荡、不稳定 | 大数据集 |
| Mini-Batch | 平衡速度与稳定 | 需要调 batch_size | **最常用** |

### 学习率衰减

```python
def learning_rate_schedule(epoch, initial_lr=0.1, decay_rate=0.95):
    """学习率衰减"""
    return initial_lr * (decay_rate ** epoch)

# 常用策略
lr_scheduler = {
    'step': lambda e: 0.1 * (0.1 ** (e // 30)),
    'cosine': lambda e: 0.5 * (1 + np.cos(np.pi * e / epochs)),
    'exponential': lambda e: 0.1 * (0.95 ** e)
}
```

## 第六部分：高级优化器

### Momentum

```python
def gradient_descent_momentum(X, y, lr=0.01, epochs=1000, momentum=0.9):
    """动量：加速收敛，减少震荡"""
    m, n = X.shape
    w = np.zeros(n)
    b = 0
    v_w = 0
    v_b = 0

    for epoch in range(epochs):
        y_pred = np.dot(X, w) + b
        dw = (1 / m) * np.dot(X.T, (y_pred - y))
        db = (1 / m) * np.sum(y_pred - y)

        # 动量更新
        v_w = momentum * v_w - lr * dw
        v_b = momentum * v_b - lr * db
        w += v_w
        b += v_b

    return w, b
```

### Adam

```python
def adam(X, y, lr=0.001, epochs=1000, beta1=0.9, beta2=0.999, eps=1e-8):
    """Adam: Adaptive Moment Estimation"""
    m, n = X.shape
    w = np.zeros(n)
    b = 0

    m_w, v_w = 0, 0  # 一阶、二阶矩估计
    m_b, v_b = 0, 0

    for epoch in range(epochs):
        y_pred = np.dot(X, w) + b
        dw = (1 / m) * np.dot(X.T, (y_pred - y))
        db = (1 / m) * np.sum(y_pred - y)

        # 更新一阶、二阶矩
        m_w = beta1 * m_w + (1 - beta1) * dw
        v_w = beta2 * v_w + (1 - beta2) * (dw ** 2)
        m_b = beta1 * m_b + (1 - beta1) * db
        v_b = beta2 * v_b + (1 - beta2) * (db ** 2)

        # 偏差校正
        m_w_hat = m_w / (1 - beta1 ** (epoch + 1))
        v_w_hat = v_w / (1 - beta2 ** (epoch + 1))

        # 更新
        w -= lr * m_w_hat / (np.sqrt(v_w_hat) + eps)
        b -= lr * m_b / (np.sqrt(v_b) + eps)

    return w, b
```

## 第八部分：名词解释

### 梯度

**定义：** 函数在某一点的方向导数，指向函数增长最快的方向。

### 损失函数

**定义：** 衡量模型预测与真实值之间差距的函数。

```python
MSE = (1/n) * Σ(y_pred - y_true)²
```

### 收敛

**定义：** 参数更新幅度越来越小，逐渐趋于稳定。

---

## ✅ 小结

1. **梯度下降**通过沿梯度方向更新参数来最小化损失
2. 三种变体：BGD、SGD、Mini-Batch GD
3. **学习率**是关键超参数，需要仔细调优
4. Adam 等自适应优化器自动调整学习率
5. 常用策略：学习率衰减、动量
