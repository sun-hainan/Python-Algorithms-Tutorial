# 梯度下降

_机器学习优化的核心算法_

---

## 第一部分：什么是梯度下降？

### 核心思想

**沿着梯度的方向，逐步找到函数的最小值。**

---

## 第二部分：代码实现

### 批量梯度下降 (BGD)

```python
def gradient_descent(X, y, lr=0.01, epochs=1000):
    m, n = X.shape
    w = np.zeros(n)
    b = 0

    for epoch in range(epochs):
        y_pred = np.dot(X, w) + b
        dw = (1 / m) * np.dot(X.T, (y_pred - y))
        db = (1 / m) * np.sum(y_pred - y)
        w -= lr * dw
        b -= lr * db

    return w, b
```

### Adam 优化器

```python
def adam(X, y, lr=0.001, epochs=1000, beta1=0.9, beta2=0.999):
    m, n = X.shape
    w = np.zeros(n)
    b = 0
    m_w, v_w = 0, 0

    for epoch in range(epochs):
        y_pred = np.dot(X, w) + b
        dw = (1 / m) * np.dot(X.T, (y_pred - y))

        m_w = beta1 * m_w + (1 - beta1) * dw
        v_w = beta2 * v_w + (1 - beta2) * (dw ** 2)
        m_w_hat = m_w / (1 - beta1 ** (epoch + 1))
        v_w_hat = v_w / (1 - beta2 ** (epoch + 1))
        w -= lr * m_w_hat / (np.sqrt(v_w_hat) + 1e-8)

    return w, b
```

## 小结

1. **梯度下降**通过沿梯度方向更新参数来最小化损失
2. 三种变体：BGD、SGD、Mini-Batch GD
3. Adam 等自适应优化器自动调整学习率
