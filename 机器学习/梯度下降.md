## 第一部分：什么是梯度？

```
梯度 = 函数在某一点上升最快的方向
负梯度 = 下降最快的方向
```

---

## 第二部分：梯度下降算法

```python
def gradient_descent(X, y, lr=0.01, epochs=1000):
    w = np.zeros(X.shape[1])

    for epoch in range(epochs):
        y_pred = np.dot(X, w)
        error = y_pred - y
        gradient = np.dot(X.T, error) / len(y)
        w -= lr * gradient

    return w
```

### 更新规则

```
θ_new = θ_old - learning_rate × ∇L(θ)
```

## 第三部分：学习率的影响

```
lr 太大：震荡、发散
lr 太小：收敛太慢
lr 合适：快速收敛
```

## 第四部分：常用优化器

### Adam

```python
def adam(X, y, lr=0.001, epochs=1000):
    w = np.zeros(X.shape[1])
    m = np.zeros_like(w)
    v = np.zeros_like(w)
    beta1, beta2 = 0.9, 0.999

    for epoch in range(epochs):
        y_pred = np.dot(X, w)
        error = y_pred - y
        gradient = np.dot(X.T, error) / len(y)

        m = beta1 * m + (1-beta1) * gradient
        v = beta2 * v + (1-beta2) * (gradient**2)
        m_hat = m / (1-beta1**epoch)
        v_hat = v / (1-beta2**epoch)
        w -= lr * m_hat / (np.sqrt(v_hat) + 1e-8)

    return w
```

## ✅ 小结

1. **梯度下降**沿负梯度方向更新参数
2. 三种变体：BGD、SGD、Mini-Batch
3. **Adam**是最常用的优化器
