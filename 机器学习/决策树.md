## 第二部分：信息熵

### 定义

```
熵 = -Σ p_i × log₂(p_i)

熵越高，数据越混乱
熵越低，数据越纯净
```

```python
def entropy(y):
    counts = np.bincount(y)
    probs = counts / len(y)
    probs = probs[probs > 0]
    return -np.sum(probs * np.log2(probs))
```

### 例子

```
全是正类: 熵 = 0.0（最纯）
一半正类一半负类: 熵 = 1.0（最混乱）
```

## 第三部分：信息增益

```
信息增益 = 分裂前熵 - 分裂后加权平均熵
```

```python
def information_gain(X, y, feature_idx, threshold):
    parent_entropy = entropy(y)
    left_mask = X[:, feature_idx] <= threshold
    right_mask = ~left_mask
    n = len(y)
    n_left = np.sum(left_mask)
    n_right = np.sum(right_mask)
    if n_left == 0 or n_right == 0:
        return 0

    child_entropy = (n_left/n)*entropy(y[left_mask]) + (n_right/n)*entropy(y[right_mask])
    return parent_entropy - child_entropy
```

## 第四部分：决策树构建

```python
class DecisionTree:
    def __init__(self, max_depth=10):
        self.max_depth = max_depth
        self.root = None

    def fit(self, X, y):
        self.root = self._build(X, y, depth=0)

    def _build(self, X, y, depth):
        if depth >= self.max_depth or len(np.unique(y)) == 1:
            return TreeNode(value=np.bincount(y).argmax())

        best_gain = -1
        best_feature = None
        best_threshold = None

        for feature_idx in range(X.shape[1]):
            for threshold in np.unique(X[:, feature_idx]):
                gain = information_gain(X, y, feature_idx, threshold)
                if gain > best_gain:
                    best_gain = gain
                    best_feature = feature_idx
                    best_threshold = threshold

        if best_gain <= 0:
            return TreeNode(value=np.bincount(y).argmax())

        left_mask = X[:, best_feature] <= best_threshold
        right_mask = ~left_mask

        return TreeNode(
            feature_idx=best_feature,
            threshold=best_threshold,
            left=self._build(X[left_mask], y[left_mask], depth+1),
            right=self._build(X[right_mask], y[right_mask], depth+1)
        )

    def predict(self, X):
        return np.array([self._traverse(x, self.root) for x in X])

    def _traverse(self, x, node):
        if node.value is not None:
            return node.value
        if x[node.feature_idx] <= node.threshold:
            return self._traverse(x, node.left)
        return self._traverse(x, node.right)
```

## ✅ 小结

1. **决策树**通过条件判断进行分类
2. 用**信息增益**选择最优分裂特征
3. 防止过拟合：限制深度、最小样本数
4. 优点：易解释、可视化
