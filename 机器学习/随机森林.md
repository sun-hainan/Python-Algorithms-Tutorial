# 随机森林

_集思广益的决策树集成_

---

## 第一部分：什么是随机森林？

### 核心思想

**让很多棵决策树一起投票决定结果。**

```
随机森林
    │
    ├── 决策树 1 ──→ 预测: A
    ├── 决策树 2 ──→ 预测: B
    ├── 决策树 3 ──→ 预测: A
    └── ...
    
最终结果: A (多数投票)
```

---

## 第二部分：两个核心技巧

### Bagging（自助聚合）

```python
def bootstrap_sample(X, y, n_samples=None):
    if n_samples is None:
        n_samples = len(X)
    indices = np.random.randint(0, n_samples, size=n_samples)
    return X[indices], y[indices]
```

## 小结

1. **随机森林**是决策树的集成
2. 两个核心技巧：**Bagging** + **特征子采样**
3. 通过多数投票减少方差，提高稳定性
