## 第一部分：感知机的生物学灵感

### 大脑中的神经元

```
        树突（接收信号）
           ↑
      ┌────┴────┐
      │         │
      │  细胞体  │
      │         │
      └────┬────┘
           ↓
        轴突（发送信号）

工作原理：
1. 树突接收其他神经元的信号
2. 如果信号足够强（超过阈值），激活
3. 通过轴突把信号传给其他神经元
```

### 感知机的诞生（1957年）

```
Frank Rosenblatt 从生物神经元得到启发
发明了感知机——第一个人工神经元模型！
```

## 第二部分：感知机的数学模型

### 结构图

```
         x₁ ── w₁ ──┐
                     │
         x₂ ── w₂ ──┼── Σ ──→ activation ──→ output y
                     │
         x₃ ── w₃ ──┤
                     │
         1  ── b ───┘
```

### 数学公式

```
Step 1: 加权求和
z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

Step 2: 激活函数（阶跃函数）
y = 1, 如果 z ≥ 0
y = 0, 如果 z < 0
```

### 代码实现

```python
import numpy as np

class Perceptron:
    """感知机"""
    def __init__(self, n_features):
        self.n_features = n_features
        # 初始化权重和偏置
        # 使用小的随机数，避免对称性
        self.weights = np.random.randn(n_features) * 0.01
        self.bias = 0.0

    def forward(self, x):
        """
        前向传播：计算感知机的输出

        x: 输入向量 (n_features,)
        返回: 0 或 1
        """
        # 加权求和
        z = np.dot(x, self.weights) + self.bias
        # 阶跃激活
        return 1 if z >= 0 else 0

    def predict(self, X):
        """
        预测多个样本

        X: 特征矩阵 (n_samples, n_features)
        返回: 预测标签数组
        """
        predictions = []
        for x in X:
            predictions.append(self.forward(x))
        return np.array(predictions)
```

## 第三部分：感知机的学习规则

### 核心思想

**"错了就改，直到全对"**

### 学习规则详解

```
如果样本预测正确（ŷ = y）：
    不更新权重

如果样本预测错误（ŷ ≠ y）：
    当 y = 1 但 ŷ = 0 时（漏检）：
        w = w + η × x      # 增加权重
        b = b + η          # 增加偏置

    当 y = 0 但 ŷ = 1 时（误报）：
        w = w - η × x      # 减少权重
        b = b - η          # 减少偏置

简写：
    w = w + η × (y - ŷ) × x
    b = b + η × (y - ŷ)
```

### 直观理解

```
假设我们要分类：●是正类，○是负类

当前决策边界（w·x + b = 0）：

    ○
         ●
    ○

情况1：● 被分类成 ○（ŷ=0, y=1）
调整：w = w + η × x（把决策边界向 ○ 方向移动）

    ○
         ● ← w 增加了
    ○

情况2：○ 被分类成 ●（ŷ=1, y=0）
调整：w = w - η × x（把决策边界向 ● 方向移动）
```

### 训练代码

```python
def train(self, X, y, learning_rate=0.1, epochs=100, verbose=True):
    """
    训练感知机

    参数:
        X: 特征矩阵 (n_samples, n_features)
        y: 标签 (n_samples,)，值为 0 或 1
        learning_rate: 学习率 η
        epochs: 迭代轮数
    """
    n_samples = len(y)

    for epoch in range(epochs):
        errors = 0  # 统计这一轮的错误数

        for x_i, y_i in zip(X, y):
            # 前向传播
            y_pred = self.forward(x_i)

            # 如果预测错误，更新参数
            if y_pred != y_i:
                errors += 1
                # 学习规则：w = w + η(y - ŷ)x
                update = learning_rate * (y_i - y_pred)
                self.weights += update * x_i
                self.bias += update

        if verbose and (epoch + 1) % 10 == 0:
            accuracy = (n_samples - errors) / n_samples
            print(f"Epoch {epoch+1:3d}: 错误数={errors}, 准确率={accuracy:.2%}")

    return self
```

## 第四部分：决策边界

### 什么是决策边界？

```
感知机的决策边界是一条直线：
w₁x₁ + w₂x₂ + ... + wₙxₙ + b = 0

在二维情况下，这是一条直线：
w₁x₁ + w₂x₂ + b = 0
即 x₂ = (-w₁x₁ - b) / w₂
```

### 例子

```python
# 逻辑AND门
X = np.array([
    [0, 0],
    [0, 1],
    [1, 0],
    [1, 1]
])
y = np.array([0, 0, 0, 1])  # AND: 只有两个都是1才输出1

p = Perceptron(n_features=2)
p.train(X, y, epochs=50, verbose=True)

print(f"权重: w1={p.weights[0]:.2f}, w2={p.weights[1]:.2f}, b={p.bias:.2f}")

# 测试
print("\n预测:")
for x in X:
    pred = p.forward(x)
    print(f"  {x} -> {pred}")
```

## 第五部分：线性可分性

### 什么是线性可分？

```
线性可分：存在一条直线（在高维是超平面），能把两类样本完全分开

    ✓ 线性可分              ✗ 非线性可分（XOR）
  ● ● ●                  ●     ●
    ╲                     ╲   ╱
     ╲_____                ● ●
                          ╱   ╲
  ○ ○ ○                  ○     ○
```

### 感知机的局限

```
1969年，Minsky 和 Papert 在《Perceptrons》中证明：
感知机只能解决线性可分问题！

具体来说：
- AND、OR 是线性可分的 → 感知机可以解决
- XOR 不是线性可分的 → 感知机无法解决
```

## 第六部分：XOR 问题详解

### 为什么感知机无法解决 XOR？

```
XOR 逻辑：
输入 (0,0) -> 输出 0
输入 (0,1) -> 输出 1
输入 (1,0) -> 输出 1
输入 (1,1) -> 输出 0

可视化：
        x₂
        ↑
      1 │  ○     ●  (0,1)   (1,1)
        │    ╲   ╱
      0 │──────●─────→ x₁
        │   ╱   ╲
      0 │  ●     ○
        │(0,0)  (1,0)

你能用一条直线把 ● 和 ○ 分开吗？
答案：不能！不存在这样的直线！
```

### 感知机的"挣扎"

```
假设我们尝试一条决策边界：

        x₂
        │
      1 │  ○     ●
        │    ╲ /
      0 │──────●───→ x₁
        │   /
      0 │  ●     ○
        │

如果边界在下方：
  (0,0) 和 (0,1) 分在一侧（但 (0,0) 应该是 ○）

如果边界在上方：
  (1,1) 和 (0,1) 分在一侧（但 (1,1) 应该是 ○）

无论怎么画，都无法同时满足所有条件！
```

## 第七部分：多层感知机解决 XOR

### 核心思想

**单层感知机画一条直线，多层可以画多条直线组合！**

### 两层感知机解决 XOR

```
XOR = (x₁ AND NOT x₂) OR (NOT x₁ AND x₂)

网络结构：
          x₁ ──┬──→ h₁ = NOT x₁ ──┐
                │                 │
          x₂ ──┼──→ h₂ = NOT x₂  │
                │                 │
                │         ┌─────── OR ──→ y
                │         │
                ├──→ h₃ = x₁ AND NOT x₂
                │
                └──→ h₄ = NOT x₁ AND x₂

等价于：
y = OR( AND(x₁, NOT x₂), AND(NOT x₁, x₂) )
```

### 代码实现

```python
class MultiLayerPerceptron:
    """多层感知机（解决XOR问题）"""
    def __init__(self):
        # 第一层：AND 门和 NOT 门
        self.and_gate = Perceptron(2)
        self.and_gate.weights = np.array([1.0, 1.0])
        self.and_gate.bias = -1.5

        # 第二层：OR 门
        self.or_gate = Perceptron(2)
        self.or_gate.weights = np.array([1.0, 1.0])
        self.or_gate.bias = -0.5

        # NOT 门：简单取反
        # NOT x = 1 - x

    def forward(self, x):
        x1, x2 = x

        # 隐藏层节点
        h1 = 1 - x1       # NOT x1
        h2 = 1 - x2       # NOT x2

        # AND 门
        and1 = self.and_gate.forward([x1, h2])  # x1 AND NOT x2
        and2 = self.and_gate.forward([h1, x2])  # NOT x1 AND x2

        # OR 门（输出）
        y = self.or_gate.forward([and1, and2])

        return y

    def predict(self, X):
        return np.array([self.forward(x) for x in X])


# 测试 XOR
mlp = MultiLayerPerceptron()
X = np.array([[0,0], [0,1], [1,0], [1,1]])
y_xor = np.array([0, 1, 1, 0])

print("XOR 测试:")
for x, y in zip(X, y_xor):
    pred = mlp.forward(x)
    print(f"  {x} -> {pred} (真实={y})")
```

## 第八部分：通用逼近定理

### 关键发现

```
1989年，George Cybenko 证明：
只需一个隐藏层的多层感知机，
就可以逼近任何连续函数！

这就是通用逼近定理（Universal Approximation Theorem）

这意味着：
- 单隐藏层 + 非线性激活函数
- 可以学习任意复杂的模式
```

### 直观理解

```
非线性函数 = 多个线性函数的叠加

    线性1   ╱
          ╱
线性2 ───╱
       ╱
      ╱ 线性3
     ╱

三个线性函数叠加 = 一个非线性函数！
```

## ✅ 小结

1. **感知机**是最简单的人工神经元模型
2. 结构：输入 → 加权求和 → 阶跃激活 → 输出
3. 学习规则：错了就改（w = w + η(y-ŷ)x）
4. 只能解决**线性可分**问题
5. **无法解决 XOR**（需要多层）
6. 多层感知机可以逼近任意函数
7. 这就是深度学习的理论基础！
