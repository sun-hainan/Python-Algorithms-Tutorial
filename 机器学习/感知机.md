# 感知机

_神经网络的最基本单元_

---

## 第一部分：什么是感知机？

### 核心思想

**感知机是最简单的神经网络模型：输入 → 加权求和 → 激活输出。**

```
感知机结构：

输入 x1 ──w1──╲
              ╲
输入 x2 ──w2──╲╲
              ╲╲
              bias ──b
               ╲╲╲
                ▼
           activation
                │
                ▼
             output y
```

## 第二部分：代码实现

```python
class Perceptron:
    def __init__(self, n_features):
        self.weights = np.random.randn(n_features) * 0.01
        self.bias = 0

    def forward(self, x):
        z = np.dot(x, self.weights) + self.bias
        return np.where(z > 0, 1, 0)

    def train(self, X, y, lr=0.1, epochs=100):
        for epoch in range(epochs):
            for x_i, y_i in zip(X, y):
                y_pred = self.forward(x_i)
                error = y_i - y_pred
                self.weights += lr * error * x_i
                self.bias += lr * error
```

## 第三部分：XOR 问题

```
XOR 是非线性可分的，感知机无法解决！

解决：多层感知机（MLP）

输入 → 隐藏层 → 输出
```

---

## 小结

1. **感知机**是最简单的神经网络模型
2. 可以实现 AND、OR 等线性可分问题
3. **无法解决 XOR 问题**（需要多层）
