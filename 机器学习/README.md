## 🔑 每章特点

### 机器学习基础
- **原理详尽**：每个算法都有数学推导和直观解释
- **代码完整**：逐行注释，可直接运行
- **对比清晰**：优缺点、适用场景一目了然

### 深度学习
- **从零实现**：NumPy 手写每个算法
- **PyTorch 应用**：工业级代码示例
- **图解过程**：数据流动、梯度传递可视化

### 生成模型
- **数学清晰**：ELBO、损失函数详细推导
- **架构拆解**：组件、工作流程完整解析
- **应用实践**：代码模板、实际案例
