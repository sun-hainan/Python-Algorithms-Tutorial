## 🎓 每章特点

### 机器学习基础（线性回归、逻辑回归等）

### 生成模型（VAE、GAN、DDPM等）
- **数学清晰**：每个公式都有直观解释
- **架构拆解**：组件、工作流程完整解析
- **代码实战**：可运行的 PyTorch 代码

## 📖 每章内容亮点

### 线性回归
```
✅ 最小二乘法：手把手推导公式，每一步都讲清楚
✅ 梯度下降：可视化理解，为什么能收敛
✅ 学习率影响：用动画说明，太大/太小/合适
✅ sklearn 实现：工业级代码
```

### 逻辑回归
```
✅ Sigmoid函数：为什么能把任意值映射到[0,1]
✅ 决策边界：图解，直观理解
✅ 交叉熵损失：为什么比MSE好
✅ 多分类：One-vs-All策略
```

### 决策树
```
✅ 信息熵：信息论视角，直观理解
✅ 信息增益：选择最佳分裂特征
✅ 过拟合：为什么会出现，怎么解决
✅ 剪枝策略：预剪枝、后剪枝
```

### 随机森林
```
✅ Bootstrap Sampling：为什么是有放回
✅ 随机特征选择：降低树之间的相关性
✅ OOB误差：不需要单独验证集
✅ 特征重要性：告诉你哪些特征最重要
```

### K-Means
```
✅ 迭代过程：每一步在做什么
✅ 肘部法则：怎么选K
✅ K-Means++初始化：避免局部最优
✅ sklearn实现：工业级代码
```

### 感知机
```
✅ 生物灵感：神经元是怎么工作的
✅ 学习规则：错了就改
✅ XOR问题：为什么单层解决不了
✅ 多层感知机：怎么解决XOR
```

### BP反向传播
```
✅ 链式法则：一步一步推导
✅ 梯度计算：每层怎么算
✅ 梯度消失/爆炸：为什么，怎么解决
✅ 完整代码：从零实现两层神经网络
```

### CNN
```
✅ 卷积操作：图解滑动窗口
✅ 局部连接 + 权重共享：为什么高效
✅ LeNet / AlexNet / ResNet：架构演进
✅ PyTorch实现：工业级代码
```

### RNN/LSTM
```
✅ 序列建模：为什么需要"记忆"
✅ 梯度消失：长期依赖问题
✅ 三个门：遗忘门、输入门、输出门
✅ GRU：LSTM的简化版
```

### Attention & Transformer
```
✅ Query-Key-Value：图书馆的比喻
✅ 多头注意力：从不同角度学习
✅ Positional Encoding：为什么需要，怎么做
✅ BERT / GPT：预训练模型基础
```

### VAE/GAN/DDPM
```
✅ 重参数化：让采样可导
✅ 对抗训练：生成器和判别器的博弈
✅ 前向/逆向过程：逐步加噪、去噪
✅ Stable Diffusion：Latent Diffusion核心
```

### LoRA
```
✅ 低秩分解：W = W₀ + BA
✅ 重参数化：推理时可以合并
✅ QLoRA：4-bit量化+LoRA
✅ 实际应用：定制AI画风、聊天机器人
```
