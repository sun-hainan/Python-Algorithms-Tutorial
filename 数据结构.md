## 🎯 学习路线

```
线性结构 ──────────────────────────────── 非线性结构
│                                           │
├─ 数组 ── O(1) 随机访问                    ├─ 树
├─ 链表 ── O(1) 插入/删除                   │   ├─ 二叉树
└─ 栈 ── LIFO（后进先出）                  │   ├─ BST
    └─ 队列 ── FIFO（先进先出）             │   └─ 堆
                                              │
└─ 哈希表 ── O(1) 查找                     └─ 图
```

## 📊 六大数据结构对比

| 数据结构 | 访问 | 搜索 | 插入 | 删除 | 特点 |
|---------|------|------|------|------|------|
| 数组 | O(1) | O(n) | O(n) | O(n) | 连续内存，高缓存命中 |
| 链表 | O(n) | O(n) | O(1) | O(1)* | 灵活但不支持随机访问 |
| 栈 | O(n) | O(n) | O(1) | O(1) | DFS，递归 |
| 队列 | O(n) | O(n) | O(1) | O(1) | BFS，任务调度 |
| 树 | O(log n) | O(log n) | O(log n) | O(log n) | 层次关系 |
| 哈希表 | - | O(1) | O(1) | O(1) | 键值对映射 |

## 🌲 二叉树遍历

```
前序（根-左-右）：1 → 2 → 4 → 5 → 3 → 6
中序（左-根-右）：4 → 2 → 5 → 1 → 6 → 3
后序（左-右-根）：4 → 5 → 2 → 6 → 3 → 1
层序（按层）：   1 → 2 → 3 → 4 → 5 → 6
```
