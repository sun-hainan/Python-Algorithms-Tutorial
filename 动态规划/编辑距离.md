# 编辑距离

> _把一个字符串变成另一个字符串最少需要几步？_

---

## 🎯 先看一个生活中的例子

### 例子：打字纠错

```
你打错了字，想把 "kitten" 改成 "sitting"：

kitten → sitten  （把 k 改成 s）
sitten → sittin  （把 e 改成 i）
sittin → sitting （在末尾加 g）

总共需要 3 步操作！

这就是"编辑距离"的概念！
```

---

## 🤔 什么是编辑距离？

### 定义

```
编辑距离（Edit Distance / Levenshtein Distance）：
把字符串 s1 变成字符串 s2 最少需要多少步操作？

允许的操作：
1. 插入（Insert）：插入一个字符
2. 删除（Delete）：删除一个字符
3. 替换（Replace）：把一个字符替换成另一个
```

### 例子

```
s1 = "horse"
s2 = "ros"

编辑过程：
horse → ros    （删除 h）
  orse → ros    （删除 o）
    rse → ros    （删除 e）

编辑距离 = 3
```

## 📐 动态规划解法

### 状态定义

```
dp[i][j] = 把 s1[0:i] 变成 s2[0:j] 的最少编辑次数

注意：s1[0:i] 表示 s1 的前 i 个字符
```

### 状态转移

```
对于 dp[i][j]：

1. 如果 s1[i-1] == s2[j-1]（最后一个字符相同）：
   dp[i][j] = dp[i-1][j-1]
   不需要操作

2. 否则（最后一个字符不同）：
   dp[i][j] = min(
       dp[i-1][j] + 1,      # 删除 s1[i-1]
       dp[i][j-1] + 1,      # 插入 s2[j-1] 到 s1
       dp[i-1][j-1] + 1     # 替换 s1[i-1] 为 s2[j-1]
   )
```

### 边界条件

```
dp[0][j] = j  # 空字符串变成 s2[0:j]，需要插入 j 次
dp[i][0] = i  # s1[0:i] 变成空字符串，需要删除 i 次
```

## 💻 代码实现

### 基础版本

```python
def edit_distance(s1, s2):
    """
    编辑距离（动态规划）
    时间复杂度：O(mn)
    空间复杂度：O(mn)
    """
    m, n = len(s1), len(s2)

    # dp[i][j] = 把 s1[0:i] 变成 s2[0:j] 的最少编辑次数
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    # 边界条件
    for i in range(m + 1):
        dp[i][0] = i  # 删除 i 个字符
    for j in range(n + 1):
        dp[0][j] = j  # 插入 j 个字符

    # 填充 dp 表
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                # 最后一个字符相同，不需要操作
                dp[i][j] = dp[i-1][j-1]
            else:
                # 三种操作取最小
                dp[i][j] = min(
                    dp[i-1][j] + 1,      # 删除
                    dp[i][j-1] + 1,      # 插入
                    dp[i-1][j-1] + 1     # 替换
                )

    return dp[m][n]


# 测试
print(edit_distance("horse", "ros"))  # 3
print(edit_distance("intention", "execution"))  # 5
```

### 过程图解

```
s1 = "horse", s2 = "ros"

构建 dp 表：

        ""  r  o  s
    ┌────────────────
 "" │ 0   1   2   3
  h │ 1   1   2   3
  o │ 2   2   1   2
  r │ 3   2   2   2
  s │ 4   3   3   2
  e │ 5   4   4   3

填充过程：

i=1, j=1 (s1[0]='h', s2[0]='r', 不同):
  dp[1][1] = min(dp[0][1]+1, dp[1][0]+1, dp[0][0]+1)
           = min(1+1, 1+1, 0+1) = 1

i=3, j=2 (s1[2]='r', s2[1]='o', 不同):
  dp[3][2] = min(dp[2][2]+1, dp[3][1]+1, dp[2][1]+1)
           = min(1+1, 2+1, 2+1) = 2

最终：dp[5][3] = 3
```

### 空间优化版本

```python
def edit_distance_optimized(s1, s2):
    """
    空间优化：只用两行
    """
    m, n = len(s1), len(s2)

    # prev 行和 curr 行
    prev = list(range(n + 1))  # dp[0][:]
    curr = [0] * (n + 1)

    for i in range(1, m + 1):
        curr[0] = i  # dp[i][0] = i
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                curr[j] = prev[j-1]
            else:
                curr[j] = min(prev[j], curr[j-1], prev[j-1]) + 1
        prev, curr = curr, prev

    return prev[n]


# 测试
print(edit_distance_optimized("horse", "ros"))  # 3
```

## 📐 路径追踪

### 如何重构编辑过程

```python
def edit_distance_with_path(s1, s2):
    """
    返回编辑距离和编辑路径
    """
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1

    # 回溯路径
    path = []
    i, j = m, n

    while i > 0 or j > 0:
        if i > 0 and j > 0 and s1[i-1] == s2[j-1]:
            path.append(f"保持: '{s1[i-1]}'")
            i -= 1
            j -= 1
        elif i > 0 and dp[i][j] == dp[i-1][j] + 1:
            path.append(f"删除: '{s1[i-1]}'")
            i -= 1
        elif j > 0 and dp[i][j] == dp[i][j-1] + 1:
            path.append(f"插入: '{s2[j-1]}'")
            j -= 1
        else:
            path.append(f"替换: '{s1[i-1]}' → '{s2[j-1]}'")
            i -= 1
            j -= 1

    path.reverse()
    return dp[m][n], path


# 测试
s1, s2 = "horse", "ros"
distance, path = edit_distance_with_path(s1, s2)
print(f"编辑距离: {distance}")
print("编辑路径:")
for step in path:
    print(f"  {step}")
```

## 🧪 经典面试题

### 题目1：单词变形

```python
def word_transformation(begin, end, word_list):
    """
    判断 begin 能否通过一次变换变成 end
    每次只能变一个字符，且变后的词必须在词表中
    """
    if end not in word_list:
        return False

    word_set = set(word_list)
    queue = [(begin, 1)]

    while queue:
        word, steps = queue.pop(0)

        if word == end:
            return steps

        # 尝试改变每个字符
        for i in range(len(word)):
            for c in 'abcdefghijklmnopqrstuvwxyz':
                new_word = word[:i] + c + word[i+1:]
                if new_word in word_set:
                    queue.append((new_word, steps + 1))
                    word_set.remove(new_word)

    return 0


# 测试
word_list = ["hot", "dot", "dog", "lot", "log", "cog"]
print(word_transformation("hit", "cog", word_list))  # 5
```

## 📊 算法对比

| 算法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 暴力枚举 | O(3^m) | O(m) |
| 递归+备忘录 | O(mn) | O(mn) |
| 动态规划 | O(mn) | O(mn) |
| 空间优化 DP | O(mn) | O(min(m,n)) |

## ✅ 本章小结

| 概念 | 说明 |
|------|------|
| 编辑距离 | 把 s1 变成 s2 的最少操作数 |
| 操作类型 | 插入、删除、替换 |
| DP 状态 | dp[i][j] = s1[0:i] 变成 s2[0:j] 的最少次数 |
| 边界条件 | dp[0][j] = j, dp[i][0] = i |
