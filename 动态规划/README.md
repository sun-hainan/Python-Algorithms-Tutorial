## 🔑 核心概念

### 动态规划三要素

```
1. 重叠子问题：子问题被多次计算
   ↓
2. 最优子结构：大问题的最优解包含小问题的最优解
   ↓
3. 状态转移：大问题和小问题的关系
```

### 解题模板

```python
# 1. 定义状态
dp[i] = ...

# 2. 状态转移
dp[i] = max(dp[i-1], ...)

# 3. 初始化
dp[0] = ...

# 4. 填表顺序
for i in range(1, n):
    dp[i] = ...
```
