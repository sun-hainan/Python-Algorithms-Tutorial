## 第一部分：什么是背包问题？

### 🎒 问题描述

**小偷入室盗窃：**
- 背包容量有限（比如 10 公斤）
- 商店里有多件宝物，每件有重量和价值
- 小偷要选择装哪些宝物，使总价值最大

### 正式定义

```
给定 n 个物品，每个物品有：
- 重量：wt[i]
- 价值：val[i]

在不超过背包容量 W 的情况下：
选择物品使总价值最大
```

---

## 第二部分：图解动态规划

### DP 思想

```
选择物品的问题
    ↓ 分解
考虑前 i 个物品，背包容量为 w
    ↓
情况1：第 i 个物品太重，装不下
  → dp[i][w] = dp[i-1][w]

情况2：第 i 个物品可以装
  → dp[i][w] = max(dp[i-1][w], dp[i-1][w-wt[i]] + val[i])
```

### 具体例子

```
背包容量 W = 10
物品：
  物品0: 重量=2, 价值=6
  物品1: 重量=4, 价值=10
  物品2: 重量=6, 价值=12

最大价值 = 22
```

## 第三部分：代码逐行解释

```python
def knapsack(W, weights, values, n):
    """
    0-1 背包问题动态规划解法

    参数:
        W: 背包容量
        weights: 物品重量列表
        values: 物品价值列表
        n: 物品数量
    返回值:
        最大价值
    """

    # -------- 创建 DP 表 --------
    # dp[i][w] 表示：考虑前 i 个物品，背包容量为 w 时的最大价值
    dp = [[0] * (W + 1) for _ in range(n + 1)]

    # -------- 填表 --------
    for i in range(1, n + 1):
        for w in range(W + 1):
            # -------- 情况1：第 i 个物品太重 --------
            if weights[i - 1] > w:
                dp[i][w] = dp[i - 1][w]

            # -------- 情况2：第 i 个物品可以装 --------
            else:
                dp[i][w] = max(
                    dp[i - 1][w],
                    dp[i - 1][w - weights[i - 1]] + values[i - 1]
                )

    return dp[n][W]
```

## 第四部分：实际运行代码

```python
def knapsack(W, weights, values, n):
    dp = [[0] * (W + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for w in range(W + 1):
            if weights[i - 1] > w:
                dp[i][w] = dp[i - 1][w]
            else:
                dp[i][w] = max(
                    dp[i - 1][w],
                    dp[i - 1][w - weights[i - 1]] + values[i - 1]
                )
    return dp[n][W]


# ========================== 测试 ==========================

print("背包容量: 10 kg\n")

items = [
    {"name": "黄金", "weight": 4, "value": 5000},
    {"name": "宝石", "weight": 3, "value": 3000},
    {"name": "现金", "weight": 2, "value": 2000},
    {"name": "电脑", "weight": 5, "value": 8000},
]

W = 10
weights = [item["weight"] for item in items]
values = [item["value"] for item in items]

max_value = knapsack(W, weights, values, len(items))
print(f"最大价值: {max_value} 元")
# 输出: 最大价值: 13000 元
```

## 第六部分：名词解释

### 动态规划 (DP)

**定义：** 把大问题分解成重叠的子问题，通过保存子问题的解避免重复计算。

### 状态转移方程

**定义：** 描述大问题和小问题关系的方程。

```python
dp[i][w] = max(
    dp[i-1][w],                           # 不选第 i 个物品
    dp[i-1][w-weights[i]] + values[i]     # 选第 i 个物品
)
```

## ✅ 小结

1. **背包问题**是组合优化的经典问题
2. **动态规划**通过保存子问题避免重复计算
3. **状态转移方程**是 DP 的核心
4. 时间复杂度 O(n × W)
