## 🎯 学习路线

```
基础算法思想
│
├─ 贪心 ──────────────────── 活动选择、哈夫曼编码
│    每步选最优，期望全局最优
│
├─ 回溯 ──────────────────── 全排列、N皇后、括号生成
│    枚举 + 剪枝 + 撤销
│
├─ 分治 ──────────────────── 归并排序、快速排序、快速幂
│    分成小问题、分别解决、合并结果
│
└─ 位运算 ────────────────── 常用技巧、LeetCode 题目
     利用二进制特性加速
```

---

## 📊 四种算法思想对比

| 算法思想 | 本质 | 适用场景 | 时间复杂度 |
|---------|------|---------|-----------|
| 贪心 | 局部最优 | 最优子结构、贪心选择性质 | 通常较快 |
| 回溯 | 枚举+剪枝 | 组合、排列、棋盘问题 | 指数级 |
| 分治 | 分而治之 | 子问题独立 | O(n log n) |
| 动态规划 | 记忆化搜索 | 子问题重叠 | 多项式级 |

## 🧠 什么时候用什么？

```
问题特征                          推荐算法
─────────────────────────────────────────────────
需要枚举所有解                     回溯
问题能分解，且子问题独立           分治
具有贪心选择性质                  贪心
子问题重叠，需要最优子结构         动态规划
```
