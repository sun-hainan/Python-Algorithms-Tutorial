## 📖 核心内容

### 二分搜索

```
核心思想：每次把搜索范围缩小一半

[1, 3, 5, 7, 9, 11, 13]
 ↑
猜中间：7 > 目标，去左边
[1, 3, 5]
 ↑
猜中间：3 < 目标，去右边
[5]
 ↑
猜中间：5 == 目标，找到！

只需 log₂7 ≈ 3 步！
```

---

## 🔑 关键点

### 二分搜索模板

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1

    while left <= right:
        mid = left + (right - left) // 2  # 防止溢出

        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1  # 未找到
```
