# 数据结构

_组织和存储数据的方式_

---

## 📋 数据结构列表

| 数据结构 | 难度 | 特点 | 应用场景 |
|----------|------|------|----------|
| 数组 | ⭐ | 连续内存，随机访问 | 基础存储 |
| 链表 | ⭐⭐ | 动态，大小不限 | 频繁插入删除 |
| 栈 | ⭐ | 后进先出 LIFO | 括号匹配，递归 |
| 队列 | ⭐ | 先进先出 FIFO | 任务调度，BFS |
| 二叉树 | ⭐⭐ | 层次结构 | 文件系统，表达式 |
| 堆 | ⭐⭐⭐ | Top-K 问题 | 优先级队列 |
