# 数据结构

_组织和存储数据的方式_

---

## 📋 数据结构列表

### 线性结构

| 数据结构 | 增 | 删 | 查 | 改 | 特点 |
|----------|----|----|----|----|------|
| 数组 | O(n) | O(n) | O(1) | O(1) | 连续内存，随机访问 |
| 链表 | O(1) | O(1) | O(n) | O(n) | 分散内存，频繁增删 |

### 特殊结构

| 数据结构 | 操作 | 特点 |
|----------|------|------|
| 栈 | push/pop O(1) | LIFO，后进先出 |
| 队列 | enqueue/dequeue O(1) | FIFO，先进先出 |
| 堆 | push/pop O(log n) | Top-K 问题 |
| 哈希表 | O(1) 平均 | 键值对，O(1) 查找 |

### 树形结构

| 数据结构 | 操作 | 特点 |
|----------|------|------|
| 二叉树 | 遍历 O(n) | 每个节点最多两个子节点 |
| 堆 | 插入/删除 O(log n) | 完全二叉树，Top-K |

## 🔑 核心概念

### 时间复杂度速查

```
数据结构      │  增    │  删    │  查    │  改
─────────────┼────────┼────────┼────────┼────────
数组          │ O(n)  │ O(n)  │ O(1)  │ O(1)
链表          │ O(1)* │ O(1)* │ O(n)  │ O(n)
栈/队列      │ O(1)  │ O(1)  │ O(n)  │ O(n)
堆           │ O(log n) │ O(log n) │ O(n) │ O(n)
哈希表       │ O(1)  │ O(1)  │ O(1)  │ O(1)
* 已知位置时
```
