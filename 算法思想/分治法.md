# 分治法

> _大事化小，小事化了——把复杂问题分成小问题，分别解决_

---

## 🎯 先看一个生活中的例子

### 军队打仗

```
问题：敌军在全国各地，我们要攻占所有敌军据点

分治策略：
1. 把全国分成几个战区
2. 每个战区派一支军队去攻打
3. 每个战区内部再分成几个省
4. 每个省派一支军队去攻打
5. ...
直到攻占每一个据点

这就是"分而治之"的思想！
```

## 🤔 什么是分治法？

### 分治法的定义

```
分治法（Divide and Conquer）：
1. 分（Divide）：把问题分解成更小的子问题
2. 治（Conquer）：递归地解决子问题
3. 合（Combine）：把子问题的解合并成原问题的解
```

### 分治法 vs 动态规划

```
分治法：
- 子问题相互独立
- 不需要保存子问题的解
- 例如：归并排序、快速幂

动态规划：
- 子问题相互重叠
- 需要保存子问题的解（备忘录）
- 例如：斐波那契、背包问题
```

## 📐 经典问题

### 问题1：归并排序（Merge Sort）

```
分治策略：
1. 分：把数组分成两半
2. 治：递归地对两半排序
3. 合：合并两个有序数组
```

```python
def merge_sort(arr):
    """归并排序"""
    if len(arr) <= 1:
        return arr

    # 分：分成两半
    mid = len(arr) // 2
    left = arr[:mid]
    right = arr[mid:]

    # 治：递归排序
    left = merge_sort(left)
    right = merge_sort(right)

    # 合：合并两个有序数组
    return merge(left, right)


def merge(left, right):
    """合并两个有序数组"""
    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    # 处理剩余元素
    result.extend(left[i:])
    result.extend(right[j:])

    return result


# 测试
arr = [38, 27, 43, 3, 9, 82, 10]
print(f"排序前: {arr}")
print(f"排序后: {merge_sort(arr)}")


# 过程图解：
# [38, 27, 43, 3, 9, 82, 10]
#      ↓ 分
# [38, 27, 43, 3]    [9, 82, 10]
#      ↓ 分                  ↓ 分
# [38, 27]  [43, 3]  [9, 82]  [10]
#      ↓ 分          ↓ 分
# [38] [27] [43] [3] [9] [82] [10]
#      ↓ 合          ↓ 合
# [27, 38]  [3, 43]  [9, 82]  [10]
#      ↓ 合                   ↓ 合
# [3, 27, 38, 43]          [9, 10, 82]
#            ↓ 合
#      [3, 9, 10, 27, 38, 43, 82]
```

### 问题2：快速排序（Quick Sort）

```
分治策略：
1. 分：选一个"基准"（pivot），把数组分成两部分
2. 治：递归地对两部分排序
3. 合：直接拼接（不需要额外合并步骤）

注意：快速排序是"原地"排序，不需要额外空间
```

```python
def quick_sort(arr):
    """快速排序"""
    if len(arr) <= 1:
        return arr

    # 分：选基准，把数组分成两部分
    pivot = arr[len(arr) // 2]  # 选择中间元素作为基准
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]

    # 治：递归排序
    # 合：拼接
    return quick_sort(left) + middle + quick_sort(right)


# 测试
arr = [38, 27, 43, 3, 9, 82, 10]
print(quick_sort(arr))


# 原地排序版本
def quick_sort_inplace(arr, low=0, high=None):
    """原地快速排序"""
    if high is None:
        high = len(arr) - 1

    if low < high:
        # 分：partition 返回基准位置
        pivot_index = partition(arr, low, high)

        # 治：递归排序两部分
        quick_sort_inplace(arr, low, pivot_index - 1)
        quick_sort_inplace(arr, pivot_index + 1, high)

    return arr


def partition(arr, low, high):
    """分区：把小于基准的放左边，大于基准的放右边"""
    pivot = arr[high]  # 选择最后一个作为基准
    i = low - 1

    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]

    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1


# 测试
arr = [38, 27, 43, 3, 9, 82, 10]
print(quick_sort_inplace(arr.copy()))
```

### 问题3：快速幂（Fast Power）

```
问题：计算 a^n（a 的 n 次方）

普通方法：O(n)
a^n = a × a × a × ... × a（乘 n 次）

快速幂：O(log n)
利用 a^n = a^(n/2) × a^(n/2)
```

```python
def power(a, n):
    """快速幂：计算 a^n"""
    if n == 0:
        return 1
    if n < 0:
        return 1 / power(a, -n)

    # 分治：a^n = a^(n//2) × a^(n//2)
    half = power(a, n // 2)

    if n % 2 == 0:
        return half * half  # n 是偶数
    else:
        return half * half * a  # n 是奇数


# 迭代版本（更高效）
def power_iterative(a, n):
    """迭代快速幂"""
    result = 1
    base = a
    exponent = n

    if exponent < 0:
        base = 1 / base
        exponent = -exponent

    while exponent > 0:
        if exponent % 2 == 1:
            result *= base
        base *= base
        exponent //= 2

    return result


# 测试
print(power(2, 10))           # 1024
print(power(3, 5))            # 243
print(power(2, -3))          # 0.125
print(power_iterative(2, 10))  # 1024
```

### 问题4：求众数（Majority Element）

```
问题：找出数组中出现超过 n/2 次的众数
（一定存在一个）

分治策略：
- 分成两半
- 分别找两边的众数
- 合并时，如果两边众数相同，就是它
- 如果不同，统计出现次数确定
```

```python
def majority_element(nums):
    """求众数（出现超过 n/2 次）"""
    return majority_recursive(nums, 0, len(nums) - 1)


def majority_recursive(nums, lo, hi):
    """分治求众数"""
    # 终止条件：只有一个元素
    if lo == hi:
        return nums[lo]

    # 分：分成两半
    mid = (hi - lo) // 2 + lo
    left = majority_recursive(nums, lo, mid)
    right = majority_recursive(nums, mid + 1, hi)

    # 合
    if left == right:
        return left

    # 两边众数不同，统计出现次数
    left_count = sum(1 for i in range(lo, hi + 1) if nums[i] == left)
    right_count = sum(1 for i in range(lo, hi + 1) if nums[i] == right)

    return left if left_count > right_count else right


# 测试
print(majority_element([3, 2, 3]))  # 3
print(majority_element([2, 2, 1, 1, 1, 2, 2]))  # 2
```

### 问题5：求最大子数组和

```python
def max_subarray(arr):
    """求最大子数组和（分治法）"""
    return max_subarray_recursive(arr, 0, len(arr) - 1)


def max_subarray_recursive(arr, low, high):
    """分治求最大子数组"""
    # 终止条件
    if low == high:
        return arr[low]

    # 分
    mid = (low + high) // 2

    # 治
    left_max = max_subarray_recursive(arr, low, mid)
    right_max = max_subarray_recursive(arr, mid + 1, high)

    # 跨中线的最大子数组
    cross_max = max_crossing_subarray(arr, low, mid, high)

    # 合：三种情况取最大
    return max(left_max, right_max, cross_max)


def max_crossing_subarray(arr, low, mid, high):
    """跨越中线的最大子数组"""
    # 左边最大
    left_sum = float('-inf')
    total = 0
    for i in range(mid, low - 1, -1):
        total += arr[i]
        left_sum = max(left_sum, total)

    # 右边最大
    right_sum = float('-inf')
    total = 0
    for i in range(mid + 1, high + 1):
        total += arr[i]
        right_sum = max(right_sum, total)

    return left_sum + right_sum


# 测试
arr = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
print(f"最大子数组和: {max_subarray(arr)}")  # 6 ([4,-1,2,1])
```

### 题目1：归并两个有序链表

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


def merge_two_lists(l1, l2):
    """合并两个有序链表"""
    if not l1:
        return l2
    if not l2:
        return l1

    if l1.val <= l2.val:
        l1.next = merge_two_lists(l1.next, l2)
        return l1
    else:
        l2.next = merge_two_lists(l1, l2.next)
        return l2
```

### 题目2：外观数列（Count and Say）

```python
def count_and_say(n):
    """外观数列"""
    if n == 1:
        return "1"

    prev = count_and_say(n - 1)
    result = []
    count = 1

    for i in range(1, len(prev)):
        if prev[i] == prev[i - 1]:
            count += 1
        else:
            result.append(str(count))
            result.append(prev[i - 1])
            count = 1

    result.append(str(count))
    result.append(prev[-1])

    return ''.join(result)


# 测试
for i in range(1, 7):
    print(f"n={i}: {count_and_say(i)}")
# n=1: 1
# n=2: 11
# n=3: 21
# n=4: 1211
# n=5: 111221
# n=6: 312211
```

## 📊 分治法复杂度分析

| 算法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 归并排序 | O(n log n) | O(n) |
| 快速排序 | O(n log n) | O(log n) |
| 快速幂 | O(log n) | O(log n) |
| 最大子数组 | O(n log n) | O(log n) |
| 众数 | O(n) | O(log n) |
