## 🔑 核心概念

### 贪心 vs 动态规划

```
贪心：每步选最优，不回头
- 优点：简单、快速
- 缺点：不一定全局最优
- 适用：有贪心选择性质的问题

动态规划：保存子问题结果，综合考虑
- 优点：全局最优
- 缺点：空间换时间
- 适用：重叠子问题 + 最优子结构
```

### 什么时候用回溯？

```
- 问题可以分解成"选择"和"不选择"
- 需要列举所有可能的解
- 解空间可以用树来表示
```
