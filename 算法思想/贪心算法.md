## 第一部分：什么是贪心算法？

### 🎯 核心思想

**每一步都选择当前最优的解，不考虑全局。**

```
贪心 = 只看眼前利益，不回头

例子：找钱问题
如果有 1元、5元、10元、50元纸币
要找 63 元
贪心策略：先拿最大的 50，再拿 10，再拿 3个 1
结果：50 + 10 + 1 + 1 + 1 = 63 ✅
```

---

## 第二部分：贪心 vs 动态规划

```
贪心：不考虑之前的选择，只看当前最优
      优点：简单、快速
      缺点：不一定能得到全局最优

动态规划：保存所有可能，综合考虑
        优点：能得到全局最优
        缺点：复杂、慢
```

## 第三部分：经典例题

### 例题1：活动选择问题

```python
def activity_selection(activities):
    sorted_activities = sorted(activities, key=lambda x: x[1])
    selected = [sorted_activities[0]]
    last_end = sorted_activities[0][1]
    for start, end in sorted_activities[1:]:
        if start >= last_end:
            selected.append((start, end))
            last_end = end
    return selected
```

### 例题2：分数背包问题

```python
def fractional_knapsack(items, capacity):
    item_values = [(name, weight, value, value/weight) for name, weight, value in items]
    sorted_items = sorted(item_values, key=lambda x: x[3], reverse=True)
    total_value = 0
    for name, weight, value, ratio in sorted_items:
        if capacity >= weight:
            total_value += value
            capacity -= weight
        else:
            total_value += value * (capacity / weight)
            break
    return total_value
```

## 第四部分：能用贪心的问题特征

```
✅ 可以用贪心的特征：
1. 最优子结构：全局最优包含局部最优
2. 贪心选择性质：局部最优能导出全局最优

❌ 不能用贪心的例子：
1. 背包问题（0-1背包）
2. 最长路径问题
```

---

## ✅ 小结

1. **贪心算法**每步选择当前最优
2. 简单高效，但不一定全局最优
3. 适用于有"贪心选择性质"的问题
