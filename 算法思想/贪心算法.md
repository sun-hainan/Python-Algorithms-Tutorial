# 贪心算法

> _走一步看一步，每步都选最优，但不一定全局最优_

---

## 🎯 先看一个生活中的例子

### 例子：找零钱

假设你是收银员，需要找零 63 元，柜子里有：
- 50 元
- 20 元
- 10 元
- 5 元
- 1 元

**贪心策略：每步都选最大的**

```
第1步：选 50 元，还需找 13 元 ✓
第2步：选 10 元，还需找 3 元  ✓
第3步：选 1 元，还需找 2 元   ✓
第4步：选 1 元，还需找 1 元   ✓
第5步：选 1 元，完成！       ✓

总共用了 5 张钞票：50+10+1+1+1 = 63 ✓
```

### 贪心 vs 动态规划

```
假设还有：11 元面额

贪心（如果用）：50 > 不用，找 20... 需要更多钞票

动态规划会全局考虑，找到最优解
但贪心在这个例子中仍然最优
```

## 🤔 什么是贪心算法？

### 贪心算法的定义

```
贪心算法（Greedy Algorithm）：
每一步都做出当前最优的选择，
期望通过局部最优达到全局最优。

关键特点：
1. 只考虑当前，不考虑未来
2. 不回溯
3. 有些问题有效，有些无效
```

### 贪心算法的适用条件

```
1. 最优子结构：全局最优解包含局部最优解
2. 贪心选择性质：可以通过局部最优达到全局最优
```

## 📐 经典问题

### 问题1：活动选择问题

```
问题描述：
- 有 n 个活动，每个活动有一个开始时间和结束时间
- 同一时间只能参加一个活动
- 找出最多能参加多少活动

贪心策略：选择最早结束的活动
```

```python
def activity_selection(activities):
    """
    活动选择问题
    activities: [(start, end), ...]
    返回：选择的活动数量
    """
    # 按结束时间排序
    activities.sort(key=lambda x: x[1])

    count = 1
    last_end = activities[0][1]

    for start, end in activities[1:]:
        if start >= last_end:  # 不重叠
            count += 1
            last_end = end

    return count


# 测试
activities = [(1, 4), (3, 5), (0, 6), (5, 7), (3, 9), (5, 9), (6, 10), (8, 11), (8, 12), (2, 14), (12, 16)]
print(f"最多参加 {activity_selection(activities)} 个活动")

# 最优选择：[(1,4), (5,7), (8,11), (12,16)] 共4个
```

### 问题2：找零钱（最优）

```python
def greedy_change(amount, coins):
    """
    贪心找零钱（前提：硬币面额是贪心最优的）
    """
    coins = sorted(coins, reverse=True)  # 从大到小
    result = []

    for coin in coins:
        while amount >= coin:
            result.append(coin)
            amount -= coin

    return result


# 测试：找 63 元
coins = [50, 20, 10, 5, 1]
print(greedy_change(63, coins))  # [50, 10, 1, 1, 1]
```

### 问题3：哈夫曼编码（贪心）

```python
import heapq
from collections import Counter

def huffman_coding(text):
    """
    哈夫曼编码 - 贪心策略
    思想：频率高的字符用短编码
    """
    # 统计频率
    freq = Counter(text)

    # 建最小堆
    heap = [[f, [c, '']] for c, f in freq.items()]
    heapq.heapify(heap)

    # 合并节点
    while len(heap) > 1:
        left = heapq.heappop(heap)
        right = heapq.heappop(heap)

        # 给左子树编码加 0，右子树加 1
        for pair in left[1:]:
            pair[1] = '0' + pair[1]
        for pair in right[1:]:
            pair[1] = '1' + pair[1]

        heapq.heappush(heap, [left[0] + right[0]] + left[1:] + right[1:])

    # 提取编码表
    codes = {pair[0]: pair[1] for pair in heap[0][1:]}
    return codes


# 测试
text = "this is an example for huffman encoding"
codes = huffman_coding(text)
print("编码表：")
for char, code in sorted(codes.items()):
    print(f"  '{char}': {code}")
```

### 问题4：分糖果（扩展）

```python
def candy(ratings):
    """
    分糖果问题
    - 每个孩子至少1颗糖
    - 评分高的孩子比邻居多

    贪心策略：从左到右一遍，从右到左一遍
    """
    n = len(ratings)
    candies = [1] * n

    # 从左到右：右边比左边高就给更多
    for i in range(1, n):
        if ratings[i] > ratings[i-1]:
            candies[i] = candies[i-1] + 1

    # 从右到左：左边比右边高就给更多
    for i in range(n-2, -1, -1):
        if ratings[i] > ratings[i+1]:
            candies[i] = max(candies[i], candies[i+1] + 1)

    return sum(candies)


# 测试
ratings = [1, 0, 2]
print(f"需要糖果: {candy(ratings)}")  # 5
```

## ⚠️ 贪心算法的局限性

### 反例：0-1 背包问题

```
问题：
- 容量 W = 50
- 物品 A：价值 60，重 10
- 物品 B：价值 100，重 30
- 物品 C：价值 120，重 50

贪心（选价值最高的）：
- 选 C（价值 120）→ 占用 50 容量
- 剩余 0 容量
- 总价值 = 120 ✗（不是最优）

最优解：
- 选 A + B = 60 + 100 = 160 ✓
```

### 反例：石子合并

```
问题：合并多堆石子，合并代价是石子数之和，求最小总代价

贪心（总是合并最小的两堆）：
- 有时会陷入局部最优
- 需要动态规划才能得到最优解
```

### 题目1：跳跃游戏

```python
def can_jump(nums):
    """
    判断是否能跳到最后
    nums[i] = 最多能跳的距离
    """
    max_reach = 0

    for i, jump in enumerate(nums):
        if i > max_reach:
            return False
        max_reach = max(max_reach, i + jump)

    return True


# 测试
print(can_jump([2, 3, 1, 1, 4]))  # True
print(can_jump([3, 2, 1, 0, 4]))  # False
```

### 题目2：钱币找零

```python
def lemonade_change(bills):
    """
    柠檬水找零
    顾客付 5/10/20 元
    需要找零

    贪心：收到 20 时，优先用 10+5 找零（保留更多 5 元）
    """
    five = ten = 0

    for bill in bills:
        if bill == 5:
            five += 1
        elif bill == 10:
            if five < 1:
                return False
            five -= 1
            ten += 1
        else:  # 20
            # 优先用 10+5（保留更多 5 元）
            if ten > 0 and five > 0:
                ten -= 1
                five -= 1
            elif five >= 3:
                five -= 3
            else:
                return False

    return True


# 测试
print(lemonade_change([5, 5, 5, 10, 20]))  # True
print(lemonade_change([10, 10]))            # True
print(lemonade_change([5, 5, 10, 10, 20])) # False
```

### 题目3：无重叠区间

```python
def erase_overlap_intervals(intervals):
    """
    移除最少的区间，使剩余区间不重叠
    """
    if not intervals:
        return 0

    intervals.sort(key=lambda x: x[1])  # 按结束时间排序

    count = 0
    last_end = intervals[0][1]

    for start, end in intervals[1:]:
        if start >= last_end:
            last_end = end
        else:
            count += 1

    return count


# 测试
intervals = [[1,2], [2,3], [3,4], [1,3]]
print(f"最少移除: {erase_overlap_intervals(intervals)}")  # 1
```
