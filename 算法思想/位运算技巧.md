## 第一部分：位运算基础

### 📊 基本操作

| 操作 | 符号 | 说明 | 示例 |
|------|------|------|------|
| 与 | & | 都是1才为1 | 5 & 3 = 1 |
| 或 | \| | 有1就为1 | 5 \| 3 = 7 |
| 异或 | ^ | 不同为1 | 5 ^ 3 = 6 |
| 左移 | << | 左移n位 | 5 << 1 = 10 |
| 右移 | >> | 右移n位 | 5 >> 1 = 2 |

## 第二部分：常见技巧

### 技巧1：交换两个数

```python
def swap(a, b):
    a ^= b
    b ^= a
    a ^= b
    return a, b
```

### 技巧2：判断奇偶

```python
def is_odd(n):
    return n & 1 == 1
```

### 技巧3：判断是否是2的幂

```python
def is_power_of_two(n):
    return n > 0 and (n & (n - 1)) == 0
```

### 技巧4：计算1的个数

```python
def count_ones(n):
    count = 0
    while n:
        n = n & (n - 1)
        count += 1
    return count
```

## 第三部分：算法应用

### 应用1：只出现一次的数字

```python
def single_number(nums):
    result = 0
    for num in nums:
        result ^= num
    return result
```

### 应用2：找出缺失的数字

```python
def missing_number(nums):
    n = len(nums)
    result = n
    for i in range(n):
        result ^= i ^ nums[i]
    return result
```

### 应用3：汉明距离

```python
def hamming_distance(x, y):
    return count_ones(x ^ y)
```

## 第四部分：位掩码

```python
# 用位掩码表示权限
READ = 1 << 0    # 001
WRITE = 1 << 1    # 010
EXECUTE = 1 << 2   # 100

user_a = READ | EXECUTE  # 101

def has_permission(user, permission):
    return user & permission == permission
```
