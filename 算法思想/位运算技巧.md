## 🤔 位运算基础

### 六种基本位运算

```
&  (AND)  ：都是1才为1     1010 & 1100 = 1000
|  (OR)   ：有1就为1       1010 | 1100 = 1110
^  (XOR)  ：不同才为1      1010 ^ 1100 = 0110
~  (NOT)  ：取反            ~1010 = 0101
<< (左移) ：向左移动       1010 << 1 = 10100
>> (右移) ：向右移动       1010 >> 1 = 101
```

## 📐 基本操作

### 1. 判断奇偶

```python
def is_odd(n):
    """判断奇偶"""
    return n & 1 == 1

# 原理：
# 偶数二进制末尾是 0
# 奇数二进制末尾是 1
# n & 1 保留最后一位

print(is_odd(5))  # True
print(is_odd(4))   # False
```

### 2. 交换两个数

```python
def swap(a, b):
    """不用临时变量交换两个数"""
    a = a ^ b
    b = a ^ b  # b = (a^b)^b = a
    a = a ^ b  # a = (a^b)^a = b
    return a, b

# 测试
print(swap(3, 5))  # (5, 3)
```

### 3. 获取第 i 位

```python
def get_bit(n, i):
    """获取 n 的第 i 位（从0开始）"""
    return (n >> i) & 1

# 测试
print(get_bit(10, 0))  # 0  (1010)
print(get_bit(10, 1))  # 1  (1010)
print(get_bit(10, 2))  # 0  (1010)
print(get_bit(10, 3))  # 1  (1010)
```

### 4. 设置第 i 位为 1

```python
def set_bit(n, i):
    """把 n 的第 i 位设为 1"""
    return n | (1 << i)

# 1 << i = 1 后面跟 i 个 0
# 1 << 2 = 100
# n | 100 把 n 的第2位变成 1

print(set_bit(0, 3))  # 8 (1000)
```

### 5. 设置第 i 位为 0

```python
def clear_bit(n, i):
    """把 n 的第 i 位设为 0"""
    return n & ~(1 << i)

# ~(1 << i) = 除了第 i 位都是 1
# n & 全1 = n（保持不变）

print(clear_bit(15, 2))  # 11 (1111 -> 1011)
```

### 6. 更新第 i 位为 v

```python
def update_bit(n, i, v):
    """把 n 的第 i 位设为 v (0或1)"""
    mask = ~(1 << i)
    return (n & mask) | (v << i)

print(update_bit(10, 1, 0))  # 8 (1010 -> 1000)
```

## 🧪 常用技巧

### 技巧1：判断是否是 2 的幂

```python
def is_power_of_two(n):
    """
    n 是 2 的幂当且仅当 n & (n-1) == 0
    """
    return n > 0 and (n & (n - 1)) == 0

# 测试
print(is_power_of_two(1))   # True  (1 = 2^0)
print(is_power_of_two(4))   # True  (4 = 2^2)
print(is_power_of_two(8))   # True  (8 = 2^3)
print(is_power_of_two(6))   # False (6 = 110)
print(is_power_of_two(0))   # False
```

**原理图解：**

```
n=4 (100), n-1=3 (011)
     100
    &011
    =000  ✓

n=6 (110), n-1=5 (101)
     110
    &101
    =100  ✗
```

### 技巧2：统计二进制中 1 的个数

```python
def count_ones(n):
    """统计 n 的二进制中 1 的个数"""
    count = 0
    while n:
        n = n & (n - 1)  # 每次消除最右边的 1
        count += 1
    return count

# 测试
print(count_ones(7))   # 3 (111)
print(count_ones(8))   # 1 (1000)
print(count_ones(255))  # 8 (11111111)
```

**原理：**

```
n=7 (111), n-1=6 (110)
n = 111 & 110 = 110, count=1
n = 110 & 101 = 100, count=2
n = 100 & 011 = 000, count=3
结束！
```

### 技巧3：获取最低位的 1

```python
def lowest_bit(n):
    """获取 n 二进制中最低位的 1"""
    return n & (-n)

# 测试
print(lowest_bit(12))  # 4 (1100 -> 0100)
print(lowest_bit(10))  # 2 (1010 -> 0010)
```

### 技巧4：两数之和（不用加法）

```python
def add(a, b):
    """不用加法实现两数相加"""
    while b != 0:
        carry = a & b  # 进位
        a = a ^ b      # 非进位和
        b = carry << 1  # 进位左移
    return a

# 测试
print(add(3, 5))   # 8
print(add(10, 20))  # 30
```

**原理：**

```
a=3 (011), b=5 (101)
第1轮：carry=001, a=110(6), b=010(2)
第2轮：carry=010, a=100(4), b=100(4)
第3轮：carry=100, a=000(0), b=1000(8)
第4轮：carry=0, 结束
结果：a=8
```

## 🧪 LeetCode 经典题

### 题目1：只出现一次的数字

```python
def single_number(nums):
    """
    除了一个数，其他都出现两次，找到它
    异或：a^a=0, a^0=a
    """
    result = 0
    for num in nums:
        result ^= num
    return result


# 测试
print(single_number([2, 2, 1]))  # 1
print(single_number([4, 1, 2, 1, 2]))  # 4
```

### 题目2：丢失的数字

```python
def missing_number(nums):
    """
    0到n中缺失一个数，找出它
    """
    n = len(nums)
    result = n
    for i in range(n):
        result ^= i ^ nums[i]
    return result


# 测试
print(missing_number([3, 0, 1]))  # 2
print(missing_number([0, 1]))      # 2
```

### 题目3：求 Hamming Distance

```python
def hamming_distance(x, y):
    """
    两个数的 Hamming Distance = 二进制不同的位数
    """
    return count_ones(x ^ y)

# 测试
print(hamming_distance(1, 4))  # 2 (01 vs 10)
```

### 题目4：颠倒二进制位

```python
def reverse_bits(n):
    """颠倒 32 位整数的二进制位"""
    result = 0
    for i in range(32):
        result = (result << 1) | ((n >> i) & 1)
    return result


# 测试
print(reverse_bits(43261596))  # 964176192
```

## 📊 位运算速度对比

| 操作 | 位运算 | 普通运算 |
|------|--------|---------|
| 乘 2^n | n << k | n * (2**k) |
| 除 2^n | n >> k | n / (2**k) |
| 判断奇偶 | n & 1 | n % 2 |
| 取绝对值 | (n ^ sign) - sign | abs(n) |
