## 第一部分：什么是冒泡排序？

### 🫧 冒泡的故事

想象水底的气泡往上冒：
- 小气泡轻，慢慢往上浮
- 大气泡重，留在下面

**冒泡排序的原理：**
- 每轮比较相邻的两个元素
- 如果顺序错了，就交换
- 每轮结束后，最大的元素会"冒泡"到最后

## 第二部分：图解过程

### 例子：将 [5, 3, 8, 1, 4] 从小到大排序

**第1轮：**
```
[5, 3, 8, 1, 4]
  ↑  ↑
  比较 5 和 3，5 > 3，交换
  ↓
[3, 5, 8, 1, 4]
     ↑  ↑
     比较 5 和 8，5 < 8，不交换
     ↓
[3, 5, 8, 1, 4]
        ↑  ↑
        比较 8 和 1，8 > 1，交换
        ↓
[3, 5, 1, 8, 4]
           ↑  ↑
           比较 8 和 4，8 > 4，交换
           ↓
[3, 5, 1, 4, 8]  ← 第1轮结束，8 到了最后

第1轮结果：[3, 5, 1, 4, 8]
最大值 8 已排序完成
```

**第2轮：**
```
[3, 5, 1, 4, 8]  ← 8 已排序，不参与
  ↑  ↑
  比较 3 和 5，3 < 5，不交换
  ↓
[3, 5, 1, 4, 8]
     ↑  ↑
     比较 5 和 1，5 > 1，交换
     ↓
[3, 1, 5, 4, 8]
        ↑  ↑
        比较 5 和 4，5 > 4，交换
        ↓
[3, 1, 4, 5, 8]  ← 第2轮结束

第2轮结果：[3, 1, 4, 5, 8]
```

**第3轮：**
```
[3, 1, 4, 5, 8]
  ↑  ↑
  比较 3 和 1，3 > 1，交换
  ↓
[1, 3, 4, 5, 8]
     ↑  ↑
     比较 3 和 4，3 < 4，不交换
     ↓
[1, 3, 4, 5, 8]
        ↑  ↑
        比较 4 和 5，4 < 5，不交换
        ↓
[1, 3, 4, 5, 8]  ← 第3轮结束

第3轮结果：[1, 3, 4, 5, 8]
```

**第4轮：**
```
[1, 3, 4, 5, 8]
  ↑  ↑
  比较 1 和 3，1 < 3，不交换
  ↓
[1, 3, 4, 5, 8]  ← 第4轮结束

排序完成：[1, 3, 4, 5, 8]
```

### 冒泡排序的核心

```
1. 从第一个元素开始
2. 比较相邻的两个元素
3. 如果左边 > 右边，交换
4. 继续下一个相邻比较
5. 每轮结束后，最大的元素到了最后
6. 重复直到没有元素需要交换
```

## 第三部分：代码逐行解释

```python
def bubble_sort(arr):
    """
    冒泡排序

    参数:
        arr: 待排序的列表（原地修改）
    返回值:
        排序后的列表
    """

    # -------- 获取数组长度 --------
    n = len(arr)

    # -------- 外层循环：控制轮数 --------
    # 第 i 轮过后，最后 i 个元素已经排好序
    # 所以只需要执行 n-1 轮
    for i in range(n - 1, 0, -1):

        # -------- 内层循环：控制每轮比较次数 --------
        # 每轮比较 arr[0] 到 arr[i]
        # arr[i] 已经是上一轮的最大值，不需要比较
        for j in range(i):

            # -------- 比较相邻元素 --------
            if arr[j] > arr[j + 1]:
                # 如果左边大于右边，交换位置
                arr[j], arr[j + 1] = arr[j + 1], arr[j]

    return arr
```

## 第四部分：实际运行代码

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n - 1, 0, -1):
        for j in range(i):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr


# ========================== 测试 ==========================

print("测试1：简单数字排序")
arr1 = [5, 3, 8, 1, 4]
print(f"  排序前: {arr1}")
print(f"  排序后: {bubble_sort(arr1)}")
# 输出: [1, 3, 4, 5, 8]

print("\n测试2：考试成绩排序")
scores = [85, 92, 78, 96, 88, 73, 91]
print(f"  排序前: {scores}")
print(f"  排序后: {bubble_sort(scores)}")
# 输出: [73, 78, 85, 88, 91, 92, 96]

print("\n测试3：已有顺序的数组")
print(f"  排序后: {bubble_sort([1, 2, 3])}")
# 输出: [1, 2, 3]

print("\n测试4：完全逆序")
print(f"  排序后: {bubble_sort([3, 2, 1])}")
# 输出: [1, 2, 3]
```

## 第六部分：排序算法对比

### 冒泡排序的特点

| 特点 | 说明 |
|------|------|
| 时间复杂度 | O(n²) |
| 空间复杂度 | O(1) |
| 稳定性 | 稳定 |
| 特点 | 简单但慢 |

### 何时使用冒泡排序？

✅ **适合使用：**
- 数据量很小（n < 100）
- 数据基本有序
- 需要稳定排序
- 教学演示

❌ **不适合使用：**
- 大规模数据排序
- 需要高性能场景

### 对比快速排序

```
数据量 n = 100 时：
- 冒泡排序：10,000 次比较
- 快速排序：664 次比较
快了近 15 倍！
```

## 第七部分：名词解释

### 外层循环 vs 内层循环

```python
for i in range(n - 1, 0, -1):      # 外层：控制轮数
    for j in range(i):            # 内层：控制每轮比较次数
```

- **外层循环**：执行 n-1 轮
- **内层循环**：每轮比较次数递减

### 交换

```python
arr[j], arr[j + 1] = arr[j + 1], arr[j]
```

Python 的多元赋值，同时交换两个元素。

### 稳定性

**稳定：** 如果两个元素相等，交换后相对位置不变。

```
原始：[2₁, 1, 2₂]
排序后：[1, 2₁, 2₂]  ← 两个 2 的顺序保持不变
```

> 冒泡排序是稳定的！
