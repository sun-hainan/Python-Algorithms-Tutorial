## 第一部分：什么是堆？

### 📦 堆的概念

**堆（Heap）** 是一种特殊的完全二叉树：
- **最大堆**：父节点的值 ≥ 子节点的值（根是最大值）
- **最小堆**：父节点的值 ≤ 子节点的值（根是最小值）

```
最大堆示例：                数组表示：
       9                      [9, 5, 6, 3, 2, 1]
      / \                   
     5   6                   
    / \  /  
   3   2 1                  

特点：根节点 9 是最大值
```

### 堆的数组表示

```
数组索引:  0   1   2   3   4   5
          ┌───┬───┬───┬───┬───┬───┐
数组:     │ 9 │ 5 │ 6 │ 3 │ 2 │ 1 │
          └───┴───┴───┴───┴───┴───┘
              │
              ↓
       9(0)
      / \
    5(1) 6(2)
    / \  /
  3(3)2(4)1(5)

规律：
- 父节点索引 i → 左子节点：2*i+1，右子节点：2*i+2
- 子节点索引 i → 父节点：(i-1)//2
```

## 第二部分：堆排序原理

### 堆排序的核心思想

1. **构建最大堆**：把数组调整成最大堆形式
2. **逐个取出**：把堆顶（最大值）与堆尾交换
3. **调整堆**：把交换后的树重新调整成最大堆
4. **重复**：直到堆大小为 1

```
Step 1: 构建最大堆
[4, 10, 3, 5, 1] → 最大堆 → [10, 5, 3, 4, 1]

Step 2: 交换堆顶和堆尾
[10, 5, 3, 4, 1] → 交换 → [1, 5, 3, 4, 10]
                                  ↑
                              已排序

Step 3: 调整堆（忽略已排序的 10）
[1, 5, 3, 4] → 调整 → [5, 4, 3, 1]

Step 4: 重复...
```

## 第三部分：代码逐行解释

```python
def heap_sort(arr):
    """
    堆排序

    参数:
        arr: 待排序的列表（原地修改）
    返回值:
        排序后的列表
    """
    n = len(arr)

    # -------- 第1步：构建最大堆 --------
    # 从最后一个有子节点的节点开始，逐个向上调整
    # 节点 i 的父节点是 (i-1)//2
    # 最后一个节点的索引是 n-1，父节点是 (n-1-1)//2 = n//2 - 1
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)

    # -------- 第2步：逐个取出堆顶 --------
    # 堆顶（最大值）与堆尾交换，然后调整堆
    for i in range(n - 1, 0, -1):
        # 交换堆顶和堆尾
        arr[0], arr[i] = arr[i], arr[0]
        # 调整堆（堆大小减小1，因为 arr[i] 已经排序完成）
        heapify(arr, i, 0)

    return arr


def heapify(arr, heap_size, root_index):
    """
    堆化：将以 root_index 为根的子树调整成最大堆

    参数:
        arr: 堆的数组表示
        heap_size: 堆的大小（用于限制调整范围）
        root_index: 当前根节点的索引
    """

    # -------- 初始化 --------
    largest = root_index      # 假设根节点是最大值
    left = 2 * root_index + 1  # 左子节点索引
    right = 2 * root_index + 2 # 右子节点索引

    # -------- 检查左子节点 --------
    # 如果左子节点存在，且比根节点大
    if left < heap_size and arr[left] > arr[largest]:
        largest = left  # 更新最大值索引

    # -------- 检查右子节点 --------
    # 如果右子节点存在，且比当前最大值大
    if right < heap_size and arr[right] > arr[largest]:
        largest = right  # 更新最大值索引

    # -------- 如果根节点不是最大值 --------
    if largest != root_index:
        # 交换根节点和最大值节点
        arr[root_index], arr[largest] = arr[largest], arr[root_index]
        # 递归调整被交换的子节点
        heapify(arr, heap_size, largest)
```

## 第四部分：实际运行代码

```python
def heap_sort(arr):
    n = len(arr)
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]
        heapify(arr, i, 0)
    return arr

def heapify(arr, heap_size, root_index):
    largest = root_index
    left = 2 * root_index + 1
    right = 2 * root_index + 2
    if left < heap_size and arr[left] > arr[largest]:
        largest = left
    if right < heap_size and arr[right] > arr[largest]:
        largest = right
    if largest != root_index:
        arr[root_index], arr[largest] = arr[largest], arr[root_index]
        heapify(arr, heap_size, largest)

## 第六部分：堆排序 vs 其他排序

### 时间复杂度

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
|------|-----------|------------|--------|
| 堆排序 | O(n log n) | O(1) ⭐ | 不稳定 |
| 快速排序 | O(n log n) | O(log n) | 不稳定 |
| 归并排序 | O(n log n) | O(n) | 稳定 |

### 堆排序的独特优势

1. **O(1) 空间复杂度** ⭐：原地排序，不需要额外空间
2. **Top-K 问题**：时间复杂度 O(n + K log n)
3. **优先队列**：天然适合实现优先队列

### 何时使用堆排序？

✅ **适合使用：**
- Top-K 问题 ⭐
- 需要 O(1) 额外空间
- 实现优先队列

## 第七部分：名词解释

### 完全二叉树

**定义：** 除最后一层外，每一层都是满的，且最后一层节点靠左。

```
完全二叉树：          非完全二叉树：
       1                     1
      / \                   / \
     2   3                 2   3
    / \                    \   
   4   5                   4    
```

### 堆化 (Heapify)

**定义：** 把一个子树调整成堆的过程。

```

### 优先队列

**定义：** 每次取出优先级最高的元素。

```python
import heapq

# Python 的 heapq 是最小堆
# 对于最大堆，可以存入负值
heap = []
heapq.heappush(heap, -5)  # 存入 5
heapq.heappush(heap, -2)  # 存入 2
heapq.heappush(heap, -8)  # 存入 8
print(-heapq.heappop(heap))  # 取出最大的：8
```

### Top-K 问题

**定义：** 找出数据中最大/最小的 K 个元素。

```python
# 普通方法：O(n log n)
sorted(arr)[-K:]

# 堆方法：O(n + K log n) ⭐ 当 K 很小时更快
```
