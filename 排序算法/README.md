# 排序算法

_将无序数据整理成有序序列_

---

## 📋 算法列表

| 算法 | 时间(平均) | 时间(最坏) | 空间 | 稳定性 |
|------|-----------|-----------|------|--------|
| 冒泡排序 | O(n²) | O(n²) | O(1) | ✅ 稳定 |
| 插入排序 | O(n²) | O(n²) | O(1) | ✅ 稳定 |
| 选择排序 | O(n²) | O(n²) | O(1) | ❌ 不稳定 |
| 快速排序 | O(n log n) | O(n²) | O(log n) | ❌ 不稳定 |
| 归并排序 | O(n log n) | O(n log n) | O(n) | ✅ 稳定 |
| 堆排序 | O(n log n) | O(n log n) | O(1) | ❌ 不稳定 |

## 🔑 核心概念

### 稳定性

```
稳定排序：相等的元素相对位置不变
[2, 1, 2'] → 排序后 → [1, 2, 2']
                        ↑  相对位置保持

不稳定排序：可能改变相对位置
[2, 1, 2'] → 排序后 → [1, 2', 2]
```

###原地排序 vs 非原地

```
原地排序：不需要额外数组
非原地排序：需要额外数组（如归并排序）
```
