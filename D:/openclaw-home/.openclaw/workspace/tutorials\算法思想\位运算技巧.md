## 第一部分：位运算基础

### 📊 基本操作

| 操作 | 符号 | 说明 | 示例 |
|------|------|------|------|
| 与 | & | 都是1才为1 | 5 & 3 = 1 |
| 或 | \| | 有1就为1 | 5 \| 3 = 7 |
| 非 | ~ | 取反 | ~5 = -6 |
| 异或 | ^ | 不同为1 | 5 ^ 3 = 6 |
| 左移 | << | 左移n位 | 5 << 1 = 10 |
| 右移 | >> | 右移n位 | 5 >> 1 = 2 |

### 二进制表示

```python
print(bin(5))    # 0b101
print(bin(3))    # 0b11
print(bin(1))    # 0b1
print(bin(0))     # 0b0

# 5 & 3 = 101 & 011 = 001 = 1
print(5 & 3)  # 1

# 5 | 3 = 101 | 011 = 111 = 7
print(5 | 3)   # 7

# 5 ^ 3 = 101 ^ 011 = 110 = 6
print(5 ^ 3)   # 6
```

## 第二部分：常见技巧

### 技巧1：交换两个数（不用临时变量）

```python
def swap(a, b):
    """异或交换"""
    a = a ^ b
    b = a ^ b
    a = a ^ b
    return a, b


print(swap(5, 3))  # (3, 5)
```

### 技巧2：判断奇偶

```python
def is_odd(n):
    return n & 1 == 1


print(is_odd(5))  # True
print(is_odd(4))  # False
```

### 技巧3：获取第 i 位

```python
def get_bit(n, i):
    """获取 n 的第 i 位（从0开始）"""
    return (n >> i) & 1


print(get_bit(5, 0))  # 1 (5 = 101)
print(get_bit(5, 1))  # 0
print(get_bit(5, 2))  # 1
```

### 技巧4：设置第 i 位为 1

```python
def set_bit(n, i):
    """将 n 的第 i 位设为 1"""
    return n | (1 << i)


print(set_bit(4, 0))  # 5 (100 -> 101)
```

### 技巧5：清除第 i 位

```python
def clear_bit(n, i):
    """将 n 的第 i 位设为 0"""
    return n & ~(1 << i)


print(clear_bit(5, 0))  # 4 (101 -> 100)
```

### 技巧6：判断是否是2的幂

```python
def is_power_of_two(n):
    """n > 0 且只有一个1"""
    return n > 0 and (n & (n - 1)) == 0


print(is_power_of_two(4))   # True
print(is_power_of_two(6))   # False
print(is_power_of_two(8))   # True
print(is_power_of_two(0))   # False
```

### 技巧7：计算一个数中1的个数

```python
def count_ones(n):
    """Brian Kernighan 算法"""
    count = 0
    while n:
        n = n & (n - 1)  # 消除最右边的1
        count += 1
    return count


print(count_ones(7))  # 3 (111)
print(count_ones(8))   # 1 (1000)
```

## 第三部分：算法应用

### 应用1：只出现一次的数字

```python
def single_number(nums):
    """
    数组中只有一个数字出现一次，其他都出现两次
    找出这个数字

    异或性质：a ^ a = 0, a ^ 0 = a
    """
    result = 0
    for num in nums:
        result ^= num
    return result


print(single_number([2, 3, 2, 4, 3]))  # 4
```

### 应用2：两数交换（不用临时变量）

```python
def swap_numbers(a, b):
    """异或交换"""
    a ^= b
    b ^= a
    a ^= b
    return a, b


print(swap_numbers(10, 20))  # (20, 10)
```

### 应用3：找出缺失的数字

```python
def missing_number(nums):
    """
    0到n中缺失了一个数字
    找出缺失的那个
    """
    n = len(nums)
    result = n  # 初始化为n
    for i in range(n):
        result ^= i ^ nums[i]
    return result


print(missing_number([0, 1, 3]))  # 2
```

### 应用4：汉明距离

```python
def hamming_distance(x, y):
    """
    两个数的二进制表示不同的位数
    """
    return count_ones(x ^ y)


print(hamming_distance(1, 4))  # 2 (001 ^ 100 = 101，有2个1)
```

### 应用5：求两个数的平均值

```python
def average(a, b):
    """防止溢出"""
    return (a & b) + (a ^ b) >> 1


print(average(10, 20))  # 15
```

## 第四部分：位掩码

### 📝 什么是位掩码？

用整数的每一位表示一种状态。

```python
# 用位掩码表示权限
READ = 1 << 0    # 001
WRITE = 1 << 1    # 010
EXECUTE = 1 << 2 # 100

# 用户A有读和执行权限
user_a = READ | EXECUTE  # 101

# 检查权限
def has_permission(user, permission):
    return user & permission == permission


print(has_permission(user_a, READ))    # True
print(has_permission(user_a, WRITE))   # False
print(has_permission(user_a, EXECUTE)) # True

# 添加权限
def add_permission(user, permission):
    return user | permission


# 删除权限
def remove_permission(user, permission):
    return user & ~permission
```

### 应用：N皇后问题（位运算优化）

```python
def solve_n_queens_bit(n):
    """
    N皇后问题（位运算优化版）
    """
    solutions = []

    def backtrack(row, cols, diag1, diag2):
        if row == n:
            solutions.append(1)
            return

        # 可用的位置
        available = (~(cols | diag1 | diag2)) & ((1 << n) - 1)

        while available:
            position = available & (-available)  # 取最低位的1
            available -= position  # 移除该位置

            backtrack(
                row + 1,
                cols | position,
                (diag1 | position) << 1,
                (diag2 | position) >> 1
            )

    backtrack(0, 0, 0, 0)
    return len(solutions)


print(solve_n_queens_bit(8))  # 92
```

## 第五部分：常见公式

| 公式 | 说明 |
|------|------|
| `n & (n-1)` | 消除最低位的1 |
| `n & (-n)` | 获取最低位的1 |
| `n \| (1 << i)` | 设置第i位为1 |
| `n & ~(1 << i)` | 清除第i位为0 |
| `(n >> i) & 1` | 获取第i位 |
| `x ^ y ^ y == x` | 异或两次得到原值 |

## 第六部分：名词解释

### 补码

```
正数的补码 = 原码
负数的补码 = 取反 + 1

Python中的负数表示：
-1 = ...111111111
-2 = ...111111110
```

### 算术右移 vs 逻辑右移

```
算术右移（Python）：保留符号位
-8 >> 1 = -4

逻辑右移：高位补0
-8 >> 1 = 大正数（取决于位数）
```
