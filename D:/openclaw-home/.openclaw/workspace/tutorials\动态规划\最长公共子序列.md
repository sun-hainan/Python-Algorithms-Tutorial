## 第一部分：什么是 LCS？

### 📝 定义

**子序列：** 从原序列中删除一些元素，剩下的元素保持原有顺序。

**最长公共子序列（LCS）：** 两个序列共同的子序列中最长的那个。

### LCS vs 子串

```
字符串1: ABCBDAB
字符串2: BDCAB

最长公共子序列(LCS): BDAB，长度为 4
最长公共子串: BAB，长度为 3
```

## 第二部分：图解 LCS

### DP 表

```
        ""  B  D  C  A  B
    ┌─────────────────────
 "" │  0  0  0  0  0  0
 A  │  0  0  0  0  1  1
 B  │  0  1  1  1  1  2
 C  │  0  1  1  2  2  2
 B  │  0  1  1  2  2  3
 D  │  0  1  2  2  2  3
 A  │  0  1  2  2  3  3
 B  │  0  1  2  2  3  4

LCS = dp[7][5] = 4
```

### 状态转移

```
dp[i][j] = 0                                   (i=0 或 j=0)
dp[i][j] = dp[i-1][j-1] + 1                    (x[i-1] == y[j-1])
dp[i][j] = max(dp[i-1][j], dp[i][j-1])        (x[i-1] != y[j-1])
```

## 第三部分：代码逐行解释

```python
def lcs(x, y):
    """
    最长公共子序列长度

    参数:
        x: 第一个序列
        y: 第二个序列
    返回值:
        LCS 长度
    """

    m, n = len(x), len(y)

    # -------- 创建 DP 表 --------
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    # -------- 填表 --------
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if x[i - 1] == y[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

    return dp[m][n]
```

## 第四部分：实际运行代码

```python
def lcs(x, y):
    m, n = len(x), len(y)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if x[i - 1] == y[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    return dp[m][n]


# ========================== 测试 ==========================

print("测试1：基本字符串")
s1 = "ABCBDAB"
s2 = "BDCAB"
print(f"  LCS长度: {lcs(s1, s2)}")
# 输出: 4
```

## 第六部分：名词解释

### 子序列 vs 子串

```
子序列：可以不连续
        "AC A" (A _ C _ _ A)

子串：必须连续
      "ABC"、"BCD"
```

### 动态规划表

**定义：** 保存子问题解的二维数组。

---

## ✅ 小结

1. **LCS** 找两个序列的最长公共子序列
2. **动态规划**通过二维表保存子问题
3. 时间复杂度 O(m × n)
4. 应用：diff比较、论文查重、DNA分析
