## 第一部分：什么是贪心算法？

### 🎯 核心思想

**每一步都选择当前最优的解，不考虑全局。**

```
贪心 = 只看眼前利益，不回头

例子：找钱问题
如果有 1元、5元、10元、50元纸币
要找 63 元
贪心策略：先拿最大的 50，再拿 10，再拿 3个 1
结果：50 + 10 + 1 + 1 + 1 = 63 ✅
```

---

## 第二部分：贪心 vs 动态规划

```
贪心：不考虑之前的选择，只看当前最优
      优点：简单、快速
      缺点：不一定能得到全局最优

动态规划：保存所有可能，综合考虑
        优点：能得到全局最优
        缺点：复杂、慢
```

## 第三部分：经典例题

### 例题1：活动选择问题

```python
def activity_selection(activities):
    """
    活动选择问题：选择最多不冲突的活动

    贪心策略：每次选择结束时间最早的活动
    """
    # 按结束时间排序
    sorted_activities = sorted(activities, key=lambda x: x[1])

    selected = [sorted_activities[0]]
    last_end = sorted_activities[0][1]

    for start, end in sorted_activities[1:]:
        if start >= last_end:
            selected.append((start, end))
            last_end = end

    return selected


# 测试
activities = [(0, 6), (1, 4), (3, 5), (5, 7), (3, 9), (5, 9), (6, 10), (8, 11), (8, 12), (2, 14), (12, 16)]
print("选择的活动：", activity_selection(activities))
```

### 例题2：哈夫曼编码

```python
import heapq

def huffman_coding(frequencies):
    """
    哈夫曼编码：贪心构建最优前缀码
    """
    heap = [[freq, [char, ""]] for char, freq in frequencies]
    heapq.heapify(heap)

    while len(heap) > 1:
        left = heapq.heappop(heap)
        right = heapq.heappop(heap)
        for pair in left[1:]:
            pair[1] = "0" + pair[1]
        for pair in right[1:]:
            pair[1] = "1" + pair[1]
        heapq.heappush(heap, [left[0] + right[0]] + left[1:] + right[1:])

    return dict(heap[0][1:])


# 测试
freqs = [("a", 45), ("b", 13), ("c", 12), ("d", 16), ("e", 9), ("f", 5)]
print("哈夫曼编码：", huffman_coding(freqs))
```

### 例题3：分数背包问题

```python
def fractional_knapsack(items, capacity):
    """
    分数背包问题：可以分割物品

    贪心策略：按单位价值从高到低排序
    """
    # 计算单位价值
    item_values = [(name, weight, value, value/weight) for name, weight, value in items]
    # 按单位价值排序
    sorted_items = sorted(item_values, key=lambda x: x[3], reverse=True)

    total_value = 0
    result = []

    for name, weight, value, ratio in sorted_items:
        if capacity >= weight:
            total_value += value
            capacity -= weight
            result.append((name, weight, value, "全拿"))
        else:
            fraction = capacity / weight
            total_value += value * fraction
            result.append((name, weight, value * fraction, f"拿{fraction:.0%}"))
            capacity = 0
            break

    return total_value, result


# 测试
items = [("金块", 10, 60), ("银块", 20, 100), ("铜块", 30, 120)]
capacity = 50
print("最大价值：", fractional_knapsack(items, capacity))
```

## 第四部分：实际应用场景

### 💰 场景1：找零钱问题

```python
def make_change(amount, coins=[50, 20, 10, 5, 1]):
    """
    用最少数量的纸币找零
    贪心策略：先用大面值的
    """
    result = {}
    for coin in coins:
        count = amount // coin
        if count > 0:
            result[coin] = count
            amount -= coin * count
    return result

print(make_change(63))
# 输出: {50: 1, 10: 1, 1: 3}
```

### 📱 场景2：任务调度

```python
def schedule_tasks(tasks):
    """
    任务调度：按截止时间排序，先做紧急的
    """
    sorted_tasks = sorted(tasks, key=lambda x: x[1])
    completed = []
    time = 0
    for name, deadline, duration in sorted_tasks:
        time += duration
        if time <= deadline:
            completed.append(name)
    return completed
```

## 第五部分：能用贪心的问题特征

```
✅ 可以用贪心的特征：
1. 最优子结构：全局最优包含局部最优
2. 贪心选择性质：局部最优能导出全局最优

❌ 不能用贪心的例子：
1. 背包问题（0-1背包）：可能需要放弃大价值物品
2. 最长路径问题：局部最优不一定全局最优
```

---

## 第六部分：名词解释

### 最优子结构

**定义：** 一个问题的最优解包含其子问题的最优解。

### 贪心选择性质

**定义：** 可以通过局部最优选择导出全局最优解。

---

## ✅ 小结

1. **贪心算法**每步选择当前最优
2. 简单高效，但不一定全局最优
3. 适用于有"贪心选择性质"的问题
4. 典型应用：活动选择、哈夫曼编码、找零钱
