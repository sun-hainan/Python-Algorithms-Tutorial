## 第一部分：栈 - 后进先出

```
栈结构：
      ┌───┐
      │ 5 │  ← 栈顶 (top)
      ├───┤
      │ 4 │
      ├───┤
      │ 3 │  ← 栈底 (bottom)
      └───┘

操作：push(入栈), pop(出栈), peek(查看栈顶)
```

---

## 第二部分：队列 - 先进先出

```
队列结构：
    入口          出口
      ┌───┬───┬───┬───┐
      │ 1 │ 2 │ 3 │ 4 │
      └───┴───┴─↑─┴───┘
      front      rear

操作：enqueue(入队), dequeue(出队)
```

## 第三部分：代码实现

```python
class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        return self.items.pop()

    def peek(self):
        return self.items[-1]


class Queue:
    def __init__(self):
        from collections import deque
        self.items = deque()

    def enqueue(self, item):
        self.items.append(item)

    def dequeue(self):
        return self.items.popleft()
```

## 第四部分：应用场景

### 📝 场景1：括号匹配（栈）

```python
def is_balanced(s):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}
    for char in s:
        if char in '([{':
            stack.append(char)
        elif char in ')]}':
            if not stack or stack[-1] != pairs[char]:
                return False
            stack.pop()
    return len(stack) == 0
```

### 🎮 场景2：BFS 遍历（队列）

```python
from collections import deque

def bfs(graph, start):
    visited = {start}
    queue = deque([start])
    while queue:
        vertex = queue.popleft()
        for neighbor in graph.get(vertex, []):
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
```

## 第五部分：名词解释

### LIFO vs FIFO

```
LIFO (Last In First Out) - 后进先出
- 像一叠盘子
- 栈是 LIFO

FIFO (First In First Out) - 先进先出
- 像排队买东西
- 队列是 FIFO
```
