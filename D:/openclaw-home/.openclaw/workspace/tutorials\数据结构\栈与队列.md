## 第一部分：栈 - 后进先出

### 📚 栈的概念

**就像一叠盘子：**
- 最后放上去的盘子，最先被拿走
- 这就是 **后进先出 (LIFO)**

```
栈结构：
      ┌───┐
      │ 5 │  ← 栈顶 (top)
      ├───┤
      │ 4 │
      ├───┤
      │ 3 │
      ├───┤
      │ 2 │
      ├───┤
      │ 1 │  ← 栈底 (bottom)
      └───┘

操作：
- push: 入栈（放上去）
- pop:  出栈（拿下来）
- peek: 查看栈顶
```

### 栈的操作图解

```
push(1), push(2), push(3)

    ┌───┐
    │ 3 │  ← top
    ├───┤
    │ 2 │
    ├───┤
    │ 1 │  ← bottom
    └───┘

pop() → 3
    ┌───┐
    │ 2 │  ← top
    ├───┤
    │ 1 │  ← bottom
    └───┘
```

## 第二部分：队列 - 先进先出

### 📚 队列的概念

**就像排队买东西：**
- 最先排队的人，最先买到
- 这就是 **先进先出 (FIFO)**

```
队列结构：
    入口          出口
      ┌───┬───┬───┬───┐
      │ 1 │ 2 │ 3 │ 4 │
      └───┴───┴─↑─┴───┘
      front      rear

操作：
- enqueue: 入队（从后面加入）
- dequeue: 出队（从前面取出）
```

### 队列的操作图解

```
enqueue(1), enqueue(2), enqueue(3)

    ┌───┬───┬───┬───┐
    │ 1 │ 2 │ 3 │   │
    └───┴───┴─↑─┴───┘
    front    rear

dequeue() → 1
    ┌───┬───┬───┬───┐
    │   │ 2 │ 3 │   │
    └───┴─↑─┴───┴───┘
         front/rear
```

## 第三部分：代码实现

### 栈的实现

```python
class Stack:
    """栈 - 用列表实现"""

    def __init__(self):
        self.items = []

    def push(self, item):
        """入栈"""
        self.items.append(item)

    def pop(self):
        """出栈"""
        if self.is_empty():
            raise IndexError("Stack is empty")
        return self.items.pop()

    def peek(self):
        """查看栈顶"""
        if self.is_empty():
            raise IndexError("Stack is empty")
        return self.items[-1]

    def is_empty(self):
        """是否为空"""
        return len(self.items) == 0

    def size(self):
        """栈大小"""
        return len(self.items)
```

### 队列的实现

```python
from collections import deque

class Queue:
    """队列 - 用 deque 实现"""

    def __init__(self):
        self.items = deque()

    def enqueue(self, item):
        """入队"""
        self.items.append(item)

    def dequeue(self):
        """出队"""
        if self.is_empty():
            raise IndexError("Queue is empty")
        return self.popleft()

    def front(self):
        """查看队首"""
        if self.is_empty():
            raise IndexError("Queue is empty")
        return self.items[0]

    def is_empty(self):
        """是否为空"""
        return len(self.items) == 0

    def size(self):
        """队列大小"""
        return len(self.items)
```

## 第五部分：实际应用场景

### 📝 场景1：括号匹配（栈）

检查表达式中的括号是否匹配。

```python
def is_balanced(s):
    """判断括号是否平衡"""
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}

    for char in s:
        if char in '([{':
            stack.append(char)
        elif char in ')]}':
            if not stack or stack[-1] != pairs[char]:
                return False
            stack.pop()

    return len(stack) == 0


print("括号匹配测试：")
print(f"  '(())'  平衡? {is_balanced('(())')}")  # True
print(f"  '([)]'  平衡? {is_balanced('([)]')}")  # False
print(f"  '{[]}'   平衡? {is_balanced('{[]}')}")   # True
```

### 🔙 场景2：浏览器前进/后退（双栈）

用两个栈实现浏览器的前进后退功能。

```python
class BrowserHistory:
    def __init__(self, homepage):
        self.back_stack = [homepage]
        self.forward_stack = []

    def visit(self, url):
        """访问新页面"""
        self.back_stack.append(url)
        self.forward_stack = []

    def back(self):
        """后退"""
        if len(self.back_stack) > 1:
            page = self.back_stack.pop()
            self.forward_stack.append(page)
            return self.back_stack[-1]
        return None

    def forward(self):
        """前进"""
        if self.forward_stack:
            page = self.forward_stack.pop()
            self.back_stack.append(page)
            return page
        return None


# 测试
browser = BrowserHistory("google.com")
browser.visit("youtube.com")
browser.visit("github.com")
print(f"当前页面: {browser.back_stack[-1]}")  # github.com
print(f"后退: {browser.back()}")              # youtube.com
print(f"后退: {browser.back()}")              # google.com
print(f"前进: {browser.forward()}")           # youtube.com
```

### 🖨️ 场景3：打印队列（队列）

打印机按顺序处理任务。

```python
class PrintQueue:
    def __init__(self):
        self.queue = Queue()

    def add_job(self, document):
        """添加打印任务"""
        self.queue.enqueue(document)
        print(f"添加任务: {document}")

    def print_next(self):
        """打印下一个任务"""
        if not self.queue.is_empty():
            job = self.queue.dequeue()
            print(f"正在打印: {job}")
        else:
            print("打印队列为空")


# 测试
pq = PrintQueue()
pq.add_job("文档1.docx")
pq.add_job("文档2.pdf")
pq.add_job("文档3.xlsx")
pq.print_next()  # 文档1.docx
pq.print_next()  # 文档2.pdf
```

### 🎮 场景4：BFS 遍历（队列）

广度优先搜索使用队列。

```python
from collections import deque

def bfs(graph, start):
    """BFS 遍历"""
    visited = {start}
    queue = deque([start])
    result = []

    while queue:
        vertex = queue.popleft()
        result.append(vertex)
        for neighbor in graph.get(vertex, []):
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)

    return result


# 测试
graph = {
    "A": ["B", "C"],
    "B": ["D", "E"],
    "C": ["F"],
    "D": [],
    "E": ["F"],
    "F": []
}
print(f"BFS 遍历: {bfs(graph, 'A')}")  # ['A', 'B', 'C', 'D', 'E', 'F']
```

## 第六部分：名词解释

### LIFO vs FIFO

```
LIFO (Last In First Out) - 后进先出
- 像一叠盘子
- 最后放上去的最先拿走
- 栈是 LIFO

FIFO (First In First Out) - 先进先出
- 像排队买东西
- 最先排队的最先买到
- 队列是 FIFO
```

### push vs pop

```
push: 入栈，把元素放到栈顶
pop:  出栈，把栈顶元素取出
```

### enqueue vs dequeue

```
enqueue: 入队，把元素放到队尾
dequeue: 出队，把队首元素取出
```

### 双端队列 (Deque)

**定义：** 两端都可以入队和出队的队列。

```python
from collections import deque

deq = deque()
deq.appendleft(1)  # 左边入队
deq.append(2)      # 右边入队
deq.popleft()      # 左边出队
deq.pop()          # 右边出队
```
