## 第一部分：什么是堆？

### 📦 堆的定义

**堆是一种特殊的完全二叉树：**
- **最大堆**：父节点的值 ≥ 子节点的值（根是最大值）
- **最小堆**：父节点的值 ≤ 子节点的值（根是最小值）

```
最大堆示例：
       9
      / \
     5   6
    / \
   3   2
```

## 第三部分：代码实现

```python
import heapq

class MinHeap:
    def __init__(self):
        self.heap = []

    def push(self, item):
        heapq.heappush(self.heap, item)

    def pop(self):
        return heapq.heappop(self.heap)

    def peek(self):
        return self.heap[0] if self.heap else None


class MaxHeap:
    def __init__(self):
        self.heap = []

    def push(self, item):
        heapq.heappush(self.heap, -item)

    def pop(self):
        return -heapq.heappop(self.heap)
```

## 第四部分：实际应用场景

### 🏆 场景1：Top-K 问题

```python
def top_k(arr, k):
    heap = []
    for num in arr[:k]:
        heapq.heappush(heap, num)
    for num in arr[k:]:
        if num > heap[0]:
            heapq.heapreplace(heap, num)
    return sorted(heap, reverse=True)
```

## 第五部分：堆排序

```python
def heap_sort(arr):
    n = len(arr)
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]
        heapify(arr, i, 0)
    return arr
```

## 第六部分：名词解释

### 完全二叉树

**定义：** 除最后一层外，每层都是满的，且最后一层节点靠左。

### 堆化 (Heapify)

**定义：** 把一个子树调整成堆的过程。

## ✅ 小结

1. **堆**是特殊的完全二叉树
2. **最大堆**：根是最大值
3. **最小堆**：根是最小值
4. 插入/弹出都是 **O(log n)**
5. 适合 **Top-K 问题**和**优先级队列**
