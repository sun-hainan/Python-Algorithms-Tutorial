## 📖 核心思想

### 贪心算法

```
每一步都选择当前最优的解
不回头，不考虑全局
```

### 回溯算法

```
尝试 → 失败 → 回退 → 尝试另一条路
像走迷宫一样
```

### 分治法

```
分解：把问题分成小问题
解决：分别解决小问题
合并：合并得到大问题的解
```

### 动态规划

```
保存子问题的解，避免重复计算
最优子结构 + 重叠子问题
```
