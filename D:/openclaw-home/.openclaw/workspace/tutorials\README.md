## 🎯 学习路径

```
入门 ──────────────────────────────────────> 进阶
│                                              │
├─ 排序：冒泡 → 插入 → 选择                     │
├─ 搜索：顺序 → 二分                           │
├─ 结构：数组链表 → 栈队列 → 树堆               │
├─ 图： BFS/DFS → Dijkstra → 其他              │
└─ DP：  Fibonacci → 背包/LCS → LIS/编辑距离   │
```
