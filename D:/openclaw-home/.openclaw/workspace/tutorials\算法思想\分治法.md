## 第一部分：什么是分治法？

### 🎯 核心思想

```
分：把大问题分解成小问题
治：分别解决小问题
合：合并小问题的解得到大问题的解
```

### 分治法 vs 动态规划

```
分治法：
- 子问题相互独立
- 不需要保存子问题的解
- 例如：归并排序、快速排序

动态规划：
- 子问题相互重叠
- 需要保存子问题的解（备忘录）
- 例如：斐波那契、背包问题
```

### 快速幂

```python
def power(base, exponent):
    """
    快速幂：计算 base^exponent

    分治：a^n = a^(n/2) * a^(n/2)
    """
    if exponent == 0:
        return 1
    if exponent < 0:
        base = 1 / base
        exponent = -exponent

    half = power(base, exponent // 2)
    if exponent % 2 == 0:
        return half * half
    else:
        return half * half * base


# 测试
print(power(2, 10))   # 1024
print(power(3, 5))    # 243
print(power(2, -3))   # 0.125
```

### 大数乘法

```python
def multiply(x, y):
    """
    大数乘法（分治法）

    将大数分成两部分，分别相乘
    """
    if len(str(x)) == 1 or len(str(y)) == 1:
        return x * y

    n = max(len(str(x)), len(str(y)))
    half = n // 2

    a, b = x // (10 ** half), x % (10 ** half)
    c, d = y // (10 ** half), y % (10 ** half)

    ac = multiply(a, c)
    bd = multiply(b, d)
    ad_bc = multiply(a + b, c + d) - ac - bd

    return ac * (10 ** (2 * half)) + ad_bc * (10 ** half) + bd


# 测试
print(multiply(1234, 5678))  # 7006652
```

### 最近点对问题

```python
import math

def closest_pair(points):
    """
    最近点对问题：找到平面中距离最近的两个点

    分治法：按x坐标分组，递归求解
    """
    def dist(p1, p2):
        return math.sqrt((p1[0] - p2[0])**2 + (p1[1] - p2[1])**2)

    points_sorted = sorted(points, key=lambda p: p[0])

    def closest(pts):
        n = len(pts)
        if n <= 3:
            # 暴力求解
            min_d = float('inf')
            result = None
            for i in range(n):
                for j in range(i + 1, n):
                    d = dist(pts[i], pts[j])
                    if d < min_d:
                        min_d = d
                        result = (pts[i], pts[j])
            return min_d, result

        # 分
        mid = n // 2
        left = pts[:mid]
        right = pts[mid:]

        # 治 + 合
        d1, pair1 = closest(left)
        d2, pair2 = closest(right)

        d = min(d1, d2)
        pair = pair1 if d1 < d2 else pair2

        # 检查跨越分界线的点
        mid_x = pts[mid][0]
        strip = [p for p in pts if abs(p[0] - mid_x) < d]

        for i in range(len(strip)):
            for j in range(i + 1, min(i + 7, len(strip))):
                d3 = dist(strip[i], strip[j])
                if d3 < d:
                    d = d3
                    pair = (strip[i], strip[j])

        return d, pair

    return closest(points_sorted)


# 测试
points = [(1, 2), (4, 6), (3, 5), (7, 1), (9, 3)]
print(closest_pair(points))
```

## 第三部分：实际应用场景

### 📊 场景1：求最大值

```python
def find_max(arr):
    """分治法求最大值"""
    if len(arr) == 1:
        return arr[0]

    mid = len(arr) // 2
    left_max = find_max(arr[:mid])
    right_max = find_max(arr[mid:])

    return max(left_max, right_max)


print(find_max([3, 1, 4, 1, 5, 9, 2, 6]))  # 9
```

### 🔍 场景2：二分搜索（特殊分治）

```python
def binary_search_divide(arr, target):
    """分治法版本的二分搜索"""
    if len(arr) == 0:
        return -1

    mid = len(arr) // 2

    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return binary_search_divide(arr[mid + 1:], target)
    else:
        result = binary_search_divide(arr[:mid], target)
        return result


arr = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(binary_search_divide(arr, 7))  # 6
```

## 第四部分：分治法模板

```python
def divide_and_conquer(problem):
    """
    分治法通用模板

    1. 分解：将问题分成子问题
    2. 解决：递归求解子问题
    3. 合并：合并子问题的解
    """

    # 递归终止条件
    if is_simple(problem):
        return solve_simple(problem)

    # 分解
    subproblems = split(problem)

    # 解决
    solutions = [divide_and_conquer(sub) for sub in subproblems]

    # 合并
    return merge(solutions)
```

## 第五部分：名词解释

### 子问题独立性

**定义：** 子问题之间不共享资源，不会相互影响。

### 递归终止条件

**定义：** 问题分解到最小规模后，需要一个明确的答案。

```python
# 归并排序的终止条件
if len(arr) <= 1:
    return arr  # 单个元素或空数组，已经有序
```

## ✅ 小结

1. **分治法** = 分解 + 解决 + 合并
2. 子问题相互独立
3. 典型应用：归并排序、快速排序、快速幂
4. 与动态规划的区别：是否需要保存子问题的解
