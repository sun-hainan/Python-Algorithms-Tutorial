## 📖 核心概念

### 稳定性
```
稳定：相同值的元素排序后相对位置不变
[2₁, 1, 2₂] → [1, 2₁, 2₂] ✅

不稳定：相同值的元素可能交换位置
[2₁, 1, 2₂] → [1, 2₂, 2₁] ❌
```

### 时间复杂度
| 复杂度 | 说明 |
|--------|------|
| O(n²) | 数据量翻倍，时间翻4倍 |
| O(n log n) | 数据量翻倍，时间略增 |
