# 编辑距离

## 第一部分：什么是编辑距离？

### 📝 定义

**编辑距离：** 把字符串 A 变成字符串 B，需要的最少编辑操作次数。

### 三种操作

| 操作 | 说明 | 代价 |
|------|------|------|
| 插入 | 插入一个字符 | 1 |
| 删除 | 删除一个字符 | 1 |
| 替换 | 把一个字符替换成另一个 | 1 |

### 例子

```
字符串 A: "horse"
字符串 B: "ros"

操作步骤:
1. horse → rorse (替换 h 为 r)
2. rorse → rose (删除 r)
3. rose → ros (删除 e)

编辑距离: 3
```

## 第二部分：代码实现

```python
def edit_distance(word1, word2):
    """
    编辑距离 - 动态规划解法

    参数:
        word1: 源字符串
        word2: 目标字符串
    返回值:
        最少编辑次数
    """
    m, n = len(word1), len(word2)

    # dp[i][j] = word1 前 i 个字符变成 word2 前 j 个字符的最少编辑次数
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    # 初始化边界
    for i in range(m + 1):
        dp[i][0] = i  # 删除 i 个字符
    for j in range(n + 1):
        dp[0][j] = j  # 插入 j 个字符

    # 填表
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = 1 + min(
                    dp[i-1][j],    # 删除
                    dp[i][j-1],    # 插入
                    dp[i-1][j-1]  # 替换
                )

    return dp[m][n]
```

## 第三部分：实际运行代码

```python
print("编辑距离测试：")
print(f"  'horse' → 'ros': {edit_distance('horse', 'ros')}")   # 3
print(f"  'kitten' → 'sitting': {edit_distance('kitten', 'sitting')}")  # 3
print(f"  'intention' → 'execution': {edit_distance('intention', 'execution')}")  # 5
```

## 第四部分：应用场景

### 🔍 拼写检查

```python
def spell_check(word, candidates):
    """找出最可能拼写正确的单词"""
    distances = [(edit_distance(word, w), w) for w in candidates]
    distances.sort()
    return [w for _, w in distances[:3]]

words = ["hello", "hallo", "help", "fell"]
print(spell_check("hallo", words))
# 输出: ['hallo', 'hello', 'fell']
```

### 📱 DNA 序列比对

```python
seq1 = "GCTGATCGATC"
seq2 = "GATCGATC"
print(f"DNA 编辑距离: {edit_distance(seq1, seq2)}")
```

## 第五部分：名词解释

### 三种编辑操作

```
插入 (Insert): 在任意位置添加一个字符
删除 (Delete): 删除任意位置的一个字符
替换 (Replace): 将一个字符替换成另一个
```

---

## ✅ 小结

1. 编辑距离衡量两个字符串的相似度
2. 三种操作：插入、删除、替换
3. 时间复杂度 O(m × n)
4. 应用：拼写检查、DNA比对
