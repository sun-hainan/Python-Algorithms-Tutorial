## 第一部分：数组 vs 链表

### 📊 一图看懂

```
数组（连续内存）：
┌───┬───┬───┬───┬───┐
│ 0 │ 1 │ 2 │ 3 │ 4 │
└───┴───┴───┴───┴───┘
  ↑                   ↑

链表（分散内存）：

  ↑           ↑           ↑
 头节点      中间        尾节点
```

## 第二部分：数组详解

### 数组的特点

| 特点 | 说明 |
|------|------|
| 内存 | 连续内存 |
| 访问 | O(1) 随机访问 |
| 插入 | O(n) 需要移动元素 |
| 删除 | O(n) 需要移动元素 |
| 大小 | 创建时固定 |

### 数组操作图解

```
访问 arr[2]：
┌───┬───┬───┬───┬───┐
│ 0 │ 1 │ 2 │ 3 │ 4 │
└───┴───┴─↑─┴───┴───┘
          ↑
    直接计算地址：base + i * size

插入（中间插入）：
┌───┬───┬───┬───┬───┐
│ A │ B │ D │ E │   │  插入 C 到位置 2
└───┴───┴─↑─┴───┴───┘
          ↓
┌───┬───┬───┬───┬───┐
│ A │ B │ C │ D │ E │  D, E 往后移动
└───┴───┴───┴───┴───┘
```

### 代码示例

```python
# 创建数组
arr = [1, 2, 3, 4, 5]

# 随机访问 O(1)
print(arr[2])  # 3

# 尾部插入 O(1)
arr.append(6)

# 中间插入 O(n)
arr.insert(2, 10)  # 在位置2插入10

# 删除 O(n)
arr.pop(2)  # 删除位置2的元素
```

## 第三部分：链表详解

### 链表的特点

| 特点 | 说明 |
|------|------|
| 内存 | 分散内存，通过指针连接 |
| 访问 | O(n) 需要遍历 |
| 插入 | O(1) 只改指针 |
| 删除 | O(1) 只改指针 |
| 大小 | 动态增长 |

### 单链表操作图解

```
链表结构：
┌───┬───┐  ┌───┬───┐  ┌───┬───┐  ┌───┬───┐
│ A │ ──┼─→│ B │ ──┼─→│ C │ ──┼─→│ D │ / │
└───┴───┘  └───┴───┘  └───┴───┘  └───┴───┘
  ↑                       ↑
 head                    tail

在 B 和 C 之间插入 X：
┌───┬───┐  ┌───┬───┐  ┌───┬───┐  ┌───┬───┐
│ A │ ──┼─→│ B │ ──┼─→│ C │ ──┼─→│ D │ / │
└───┴───┘  └───┴─↑─┘  └─↑─┴───┘  └───┴───┘
               │    │
               └──X─┘

插入后：
┌───┬───┐  ┌───┬───┐  ┌───┬───┐  ┌───┬───┐
│ A │ ──┼─→│ B │ ──┼─→│ X │ ──┼─→│ C │ ──┼─→│ D │ / │
└───┴───┘  └───┴───┘  └───┴───┘  └───┴───┘
```

### 代码实现

```python
class ListNode:
    """链表节点"""
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


def create_linked_list(values):
    """从列表创建链表"""
    if not values:
        return None
    head = ListNode(values[0])
    current = head
    for val in values[1:]:
        current.next = ListNode(val)
        current = current.next
    return head


def print_linked_list(head):
    """打印链表"""
    values = []
    while head:
        values.append(str(head.val))
        head = head.next
    print(" -> ".join(values) + " -> None")


def reverse_linked_list(head):
    """反转链表"""
    prev = None
    curr = head
    while curr:
        next_node = curr.next  # 保存下一个节点
        curr.next = prev       # 反转指针
        prev = curr            # prev 前移
        curr = next_node       # curr 前移
    return prev


# ========================== 测试 ==========================

# 创建链表：1 -> 2 -> 3 -> 4 -> 5
head = create_linked_list([1, 2, 3, 4, 5])
print("原链表：")
print_linked_list(head)
# 输出: 1 -> 2 -> 3 -> 4 -> 5 -> None

# 反转链表
print("\n反转后：")
new_head = reverse_linked_list(head)
print_linked_list(new_head)
# 输出: 5 -> 4 -> 3 -> 2 -> 1 -> None
```

## 第四部分：对比与选择

### 数组 vs 链表

| 操作 | 数组 | 链表 | 赢家 |
|------|------|------|------|
| 随机访问 | O(1) | O(n) | 数组 ⭐ |
| 头部插入 | O(n) | O(1) | 链表 ⭐ |
| 尾部插入 | O(1) | O(1) | 平局 |
| 中间插入 | O(n) | O(1)* | 链表 ⭐ |
| 内存使用 | 紧凑 | 有指针开销 | 数组 |
| 缓存友好 | ✅ 是 | ❌ 否 | 数组 |

*链表中间插入需要先遍历找到位置，O(n)

### 何时使用数组？

✅ **适合使用：**
- 需要频繁随机访问
- 数据量固定
- 需要利用 CPU 缓存

### 何时使用链表？

✅ **适合使用：**
- 需要频繁插入/删除
- 数据量不确定
- 不需要随机访问

## 第五部分：实际应用场景

### 📱 场景1：播放器播放列表

需要频繁添加/删除歌曲。

```python
# 链表实现播放列表
class Playlist:
    def __init__(self):
        self.head = None
        self.tail = None

    def add_song(self, name):
        """添加歌曲到末尾"""
        new_node = ListNode(name)
        if not self.head:
            self.head = new_node
            self.tail = new_node
        else:
            self.tail.next = new_node
            self.tail = new_node

    def remove_song(self, name):
        """删除歌曲"""
        if not self.head:
            return
        if head.val == name:
            self.head = head.next
            return
        prev = head
        curr = head.next
        while curr:
            if curr.val == name:
                prev.next = curr.next
                break
            prev = curr
            curr = curr.next
```

### 🎵 场景2：浏览器前进/后退

用两个栈实现前进后退。

```python
class BrowserHistory:
    def __init__(self, homepage):
        self.back_stack = [homepage]  # 后退栈
        self.forward_stack = []        # 前进栈

    def visit(self, url):
        """访问新页面"""
        self.back_stack.append(url)
        self.forward_stack = []  # 清空前进栈

    def back(self):
        """后退"""
        if len(self.back_stack) > 1:
            page = self.back_stack.pop()
            self.forward_stack.append(page)
            return self.back_stack[-1]
        return None
```

### 🏦 场景3：任务队列

按顺序处理任务。

```python
class TaskQueue:
    def __init__(self):
        self.tasks = []  # 用列表模拟队列

    def add_task(self, task):
        """添加任务"""
        self.tasks.append(task)

    def get_next_task(self):
        """获取下一个任务"""
        if self.tasks:
            return self.tasks.pop(0)  # 从头部取出
        return None
```

## 第六部分：名词解释

### 随机访问

**定义：** 通过索引直接访问任意元素，不需要遍历。

```python
# 数组支持随机访问 O(1)
arr = [10, 20, 30, 40, 50]
print(arr[3])  # 40，直接计算地址访问
```

### 指针/引用

**定义：** 存储另一个节点的地址。

```python
class Node:
    def __init__(self, val):
        self.val = val
        self.next = None  # 指针指向下一个节点

node1 = Node(1)
node2 = Node(2)
node1.next = node2  # node1.next 指向 node2
```

### 头节点/尾节点

**定义：**
- **头节点**：链表的第一个节点
- **尾节点**：链表的最后一个节点

```python
head = Node(1)  # 头节点
# ... 添加更多节点
tail.next = Node(5)  # 添加尾节点
```

### 虚拟头节点

**定义：** 在链表头部添加一个空节点，简化边界处理。

```python
class DummyHeadList:
    def __init__(self):
        self.dummy = ListNode(0)  # 虚拟头节点
        self.size = 0

    def add(self, index, val):
        prev = self.dummy
        for _ in range(index):
            prev = prev.next
        prev.next = ListNode(val)
        self.size += 1
```

## ✅ 小结

| 对比项 | 数组 | 链表 |
|--------|------|------|
| 内存 | 连续 | 分散 |
| 访问 | O(1) ⭐ | O(n) |
| 插入/删除 | O(n) | O(1) ⭐ |
| 内存开销 | 小 | 大（有指针）|

**选择建议：**
- 需要频繁访问 → 用数组
- 需要频繁插入/删除 → 用链表
