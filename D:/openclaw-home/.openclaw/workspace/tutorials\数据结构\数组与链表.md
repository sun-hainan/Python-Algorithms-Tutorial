## 第一部分：数组 vs 链表

```
数组（连续内存）：
┌───┬───┬───┬───┬───┐
│ 0 │ 1 │ 2 │ 3 │ 4 │
└───┴───┴───┴───┴───┘

链表（分散内存）：
┌───┬───┐  ┌───┬───┐  ┌───┬───┐
│ 1 │ ──┼─→│ 2 │ ──┼─→│ 3 │ / │
└───┴───┘  └───┴───┘  └───┴───┘
```

---

## 第二部分：数组详解

| 特点 | 说明 |
|------|------|
| 内存 | 连续内存 |
| 访问 | O(1) 随机访问 |
| 插入 | O(n) 需要移动元素 |
| 删除 | O(n) 需要移动元素 |

---

## 第三部分：链表详解

| 特点 | 说明 |
|------|------|
| 内存 | 分散内存，通过指针连接 |
| 访问 | O(n) 需要遍历 |
| 插入 | O(1) 只改指针 |
| 删除 | O(1) 只改指针 |

---

## 第四部分：代码实现

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


def reverse_linked_list(head):
    """反转链表"""
    prev = None
    curr = head
    while curr:
        next_node = curr.next
        curr.next = prev
        prev = curr
        curr = next_node
    return prev
```

---

## 第五部分：对比与选择

| 操作 | 数组 | 链表 | 赢家 |
|------|------|------|------|
| 随机访问 | O(1) | O(n) | 数组 ⭐ |
| 头部插入 | O(n) | O(1) | 链表 ⭐ |

## 第七部分：名词解释

### 随机访问

**定义：** 通过索引直接访问任意元素，不需要遍历。
