## 第一部分：什么是哈希表？

### 🎯 核心思想

**通过哈希函数，把键映射到数组的某个位置，实现 O(1) 查找。**

```
键 (key)  →  哈希函数  →  数组索引  →  值 (value)

"Alice"  →  hash("Alice")  →     3     →  "alice@email.com"
```

---

## 第二部分：哈希函数

### 📝 什么是哈希函数？

把任意长度的输入转换为固定长度的输出。

```python
# 简单的哈希函数
def hash_func(key, size):
    return hash(key) % size

# 测试
print(hash_func("apple", 10))  # 可能是 3
print(hash_func("banana", 10)) # 可能是 7
print(hash_func("cherry", 10)) # 可能是 1
```

### 常见的哈希函数

```
1. 除法哈希：h(key) = key % m
2. 乘法哈希：h(key) = floor(m * (key * A % 1))
3. 字符串哈希：把字符串转成数字后取模
```

## 第三部分：冲突处理

### ⚠️ 哈希冲突

不同的键可能映射到同一个位置。

```
"apple" → hash → 3
"grape" → hash → 3  # 冲突！
```

### 解决方法

#### 方法1：链地址法（拉链法）

```
数组位置 3: [apple] -> [grape] -> [orange]
```

```python
class HashTable:
    def __init__(self, size=10):
        self.size = size
        self.buckets = [[] for _ in range(size)]

    def hash(self, key):
        return hash(key) % self.size

    def put(self, key, value):
        index = self.hash(key)
        for i, (k, v) in enumerate(self.buckets[index]):
            if k == key:
                self.buckets[index][i] = (key, value)
                return
        self.buckets[index].append((key, value))

    def get(self, key):
        index = self.hash(key)
        for k, v in self.buckets[index]:
            if k == key:
                return v
        return None
```

#### 方法2：开放地址法

```
冲突时，找下一个空位置
线性探测：h(key) + 1, h(key) + 2, ...
```

```python
class OpenAddressHashTable:
    def __init__(self, size=10):
        self.size = size
        self.keys = [None] * size
        self.values = [None] * size

    def put(self, key, value):
        index = self.hash(key)
        while self.keys[index] is not None:
            if self.keys[index] == key:
                self.values[index] = value
                return
            index = (index + 1) % self.size
        self.keys[index] = key
        self.values[index] = value

    def get(self, key):
        index = self.hash(key)
        start = index
        while self.keys[index] is not None:
            if self.keys[index] == key:
                return self.values[index]
            index = (index + 1) % self.size
            if index == start:
                break
        return None
```

## 第四部分：Python 的字典

### 📚 Python 内置哈希表

```python
# 创建
hash_table = {}

# 添加
hash_table["apple"] = 3.99
hash_table["banana"] = 1.99
hash_table["cherry"] = 2.99

# 查找 O(1)
print(hash_table["apple"])  # 3.99

# 检查存在
print("apple" in hash_table)  # True

# 删除
del hash_table["banana"]

# 遍历
for key, value in hash_table.items():
    print(f"{key}: {value}")
```

### 🔍 场景2：两数之和

```python
def two_sum(nums, target):
    """
    找出数组中两个数的和等于target
    时间复杂度: O(n)
    """
    seen = {}  # 哈希表存储已见过的数

    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i

    return []


print(two_sum([2, 7, 11, 15], 9))  # [0, 1]
```

### 📝 场景3：词频统计

```python
def word_frequency(text):
    """统计单词出现频率"""
    freq = {}
    for word in text.split():
        freq[word] = freq.get(word, 0) + 1
    return freq


text = "apple banana apple orange banana apple"
print(word_frequency(text))
# {'apple': 3, 'banana': 2, 'orange': 1}
```

### 🗓️ 场景4：LRU 缓存

```python
from collections import OrderedDict

class LRUCache:
    """最近最少使用缓存"""
    def __init__(self, capacity):
        self.capacity = capacity
        self.cache = OrderedDict()

    def get(self, key):
        if key in self.cache:
            self.cache.move_to_end(key)
            return self.cache[key]
        return -1

    def put(self, key, value):
        if key in self.cache:
            self.cache.move_to_end(key)
        self.cache[key] = value
        if len(self.cache) > self.capacity:
            self.cache.popitem(last=False)
```

## 第七部分：名词解释

### 哈希函数

**定义：** 把任意长度的输入转换为固定长度输出的函数。

### 冲突

**定义：** 不同的输入经过哈希函数得到相同的输出。

### 负载因子

**定义：** 哈希表中元素数量 / 哈希表大小。

```
负载因子 = n / m

负载因子过大 → 冲突增多 → 性能下降
通常当负载因子 > 0.75 时，需要扩容
```
