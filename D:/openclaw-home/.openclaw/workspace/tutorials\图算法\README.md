## 📖 核心概念

### 图的表示

```
邻接表（适合稀疏图）：
{
    'A': ['B', 'C'],
    'B': ['A', 'D'],
    'C': ['A'],
    'D': ['B']
}

邻接矩阵（适合稠密图）：
    A  B  C  D
 A   0  1  1  0
 B   1  0  0  1
 C   1  0  0  0
 D   0  1  0  0
```
