## 第一部分：二叉树的概念

```
二叉树示例：

           A
          / \
         B   C
        / \   \
       D   E   F
```

### 重要术语

- **根节点**：树的顶部节点（A）
- **叶子节点**：没有孩子的节点（D, E, F）

---

## 第二部分：四种遍历方式

| 遍历 | 顺序 | 应用场景 |
|------|------|----------|
| 前序 | 根 → 左 → 右 | 文件系统 |
| 中序 | 左 → 根 → 右 | BST 排序 |
| 后序 | 左 → 右 → 根 | 依赖关系 |
| 层序 | 按层遍历 | 最短路径 |

### 图解

```
           A
          / \
         B   C
        / \   \
       D   E   F

前序: A B D E C F
中序: D B E A C F
后序: D E B F C A
层序: A B C D E F
```

## 第三部分：代码实现

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


def preorder(root):
    if not root:
        return []
    return [root.val] + preorder(root.left) + preorder(root.right)


def inorder(root):
    if not root:
        return []
    return inorder(root.left) + [root.val] + inorder(root.right)


def postorder(root):
    if not root:
        return []
    return postorder(root.left) + postorder(root.right) + [root.val]


def level_order(root):
    if not root:
        return []
    from collections import deque
    queue = deque([root])
    result = []
    while queue:
        node = queue.popleft()
        result.append(node.val)
        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)
    return result
```
