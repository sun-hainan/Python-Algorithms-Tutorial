## 第一部分：二叉树的概念

### 🌳 什么是二叉树？

**每个节点最多有两个孩子的树结构**

```
二叉树示例：

           A
          / \
         B   C        ← A 是根节点
        / \   \       ← B 是 A 的左孩子
       D   E   F      ← D, E 是 B 的孩子
                          F 是 C 的右孩子
```

### 重要术语

```
树的层级（从1开始）：
           A        ← 第1层
          / \
         B   C     ← 第2层
        / \   \
       D   E   F   ← 第3层

- 根节点：树的顶部节点（A）
- 叶子节点：没有孩子的节点（D, E, F）
- 深度：从根到该节点的路径长度（A的深度=0）
- 高度：从该节点到最深叶子节点的路径长度
```

## 第二部分：二叉树的遍历

### 四种遍历方式

| 遍历 | 顺序 | 应用场景 |
|------|------|----------|
| 前序 | 根 → 左 → 右 | 文件系统 |
| 中序 | 左 → 根 → 右 | BST 排序 |
| 后序 | 左 → 右 → 根 | 依赖关系 |
| 层序 | 按层遍历 | 最短路径 |

### 图解遍历

```
二叉树：
           A
          / \
         B   C
        / \   \
       D   E   F

前序 (Preorder): A B D E C F
  → 根在前，先访问根

中序 (Inorder): D B E A C F
  → 左子树 → 根 → 右子树
  → BST 遍历后是有序的！

后序 (Postorder): D E B F C A
  → 左子树 → 右子树 → 根

层序 (Level): A B C D E F
  → 按层从左到右
```

## 第三部分：代码实现

```python
class TreeNode:
    """二叉树节点"""
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left    # 左孩子
        self.right = right  # 右孩子


def build_tree():
    """构建示例二叉树"""
    #       A
    #      / \
    #     B   C
    #    / \   \
    #   D   E   F
    return TreeNode('A',
        left=TreeNode('B',
            left=TreeNode('D'),
            right=TreeNode('E')
        ),
        right=TreeNode('C',
            right=TreeNode('F')
        )
    )


def preorder(root):
    """前序遍历：根 → 左 → 右"""
    if not root:
        return []
    return [root.val] + preorder(root.left) + preorder(root.right)


def inorder(root):
    """中序遍历：左 → 根 → 右"""
    if not root:
        return []
    return inorder(root.left) + [root.val] + inorder(root.right)


def postorder(root):
    """后序遍历：左 → 右 → 根"""
    if not root:
        return []
    return postorder(root.left) + postorder(root.right) + [root.val]


def level_order(root):
    """层序遍历：按层从左到右"""
    if not root:
        return []

    from collections import deque
    queue = deque([root])
    result = []

    while queue:
        node = queue.popleft()
        result.append(node.val)
        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)

    return result
```

## 第四部分：实际运行代码

```python
# 构建二叉树
root = build_tree()

print("四种遍历结果：")
print(f"  前序: {preorder(root)}")    # ['A', 'B', 'D', 'E', 'C', 'F']
print(f"  中序: {inorder(root)}")    # ['D', 'B', 'E', 'A', 'C', 'F']
print(f"  后序: {postorder(root)}")  # ['D', 'E', 'B', 'F', 'C', 'A']
print(f"  层序: {level_order(root)}") # ['A', 'B', 'C', 'D', 'E', 'F']
```

## 第五部分：实际应用场景

### 📁 场景1：文件系统结构

文件夹和文件的层次结构。

```python
# 文件系统
file_system = {
    "name": "根目录",
    "type": "folder",
    "children": [
        {
            "name": "文档",
            "type": "folder",
            "children": [
                {"name": "简历.docx", "type": "file"},
                {"name": "报告.pdf", "type": "file"},
            ]
        },
        {
            "name": "图片",
            "type": "folder",
            "children": [
                {"name": "头像.png", "type": "file"},
            ]
        }
    ]
}

def print_tree(node, indent=0):
    """打印文件树"""
    print("  " * indent + f"📁 {node['name']}")
    if "children" in node:
        for child in node["children"]:
            print_tree(child, indent + 1)
```

### 📊 场景2：表达式树

数学表达式可以用二叉树表示。

```python
# 表达式 (3 + 4) * 2
#
#        *
#       / \
#      +   2
#     / \
#    3   4

expr_tree = TreeNode('*',
    left=TreeNode('+',
        left=TreeNode(3),
        right=TreeNode(4)
    ),
    right=TreeNode(2)
)

def eval_tree(root):
    """计算表达式树的值"""
    if isinstance(root.val, (int, float)):
        return root.val
    if root.val == '+':
        return eval_tree(root.left) + eval_tree(root.right)
    if root.val == '-':
        return eval_tree(root.left) - eval_tree(root.right)
    if root.val == '*':
        return eval_tree(root.left) * eval_tree(root.right)
    if root.val == '/':
        return eval_tree(root.left) / eval_tree(root.right)

print(f"(3 + 4) * 2 = {eval_tree(expr_tree)}")  # 14
```

### 🎯 场景3：BST 二叉搜索树

快速查找、插入、删除。

```python
class BSTNode:
    """二叉搜索树节点"""
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None


def insert_bst(root, val):
    """插入节点"""
    if not root:
        return BSTNode(val)
    if val < root.val:
        root.left = insert_bst(root.left, val)
    else:
        root.right = insert_bst(root.right, val)
    return root


def search_bst(root, target):
    """搜索节点"""
    if not root or root.val == target:
        return root
    if target < root.val:
        return search_bst(root.left, target)
    return search_bst(root.right, target)


# 测试 BST
values = [5, 3, 7, 1, 4, 6, 8]
bst_root = None
for v in values:
    bst_root = insert_bst(bst_root, v)

print(f"BST 中序遍历（有序）: {inorder(bst_root)}")
# 输出: [1, 3, 4, 5, 6, 7, 8]

print(f"搜索 6: {'找到' if search_bst(bst_root, 6) else '未找到'}")
```

## 第六部分：遍历对比

| 遍历 | 顺序 | 递归实现 | 空间复杂度 |
|------|------|----------|------------|
| 前序 | 根 → 左 → 右 | 简单 | O(h) |
| 中序 | 左 → 根 → 右 | 简单 | O(h) |
| 后序 | 左 → 右 → 根 | 简单 | O(h) |
| 层序 | 按层 | 需要队列 | O(w) |

*h = 树的高度, w = 树的最大宽度

## 第七部分：名词解释

### 递归遍历

**定义：** 函数调用自己遍历子树。

```python
def preorder(root):
    if not root:
        return
    print(root.val)      # 访问根
    preorder(root.left)   # 遍历左子树
    preorder(root.right)  # 遍历右子树
```

### 栈帧 (Stack Frame)

**定义：** 递归调用时保存的局部变量和返回地址。

```python
# 递归深度过大可能导致栈溢出
# Python 默认递归深度限制：约 1000
import sys
print(sys.getrecursionlimit())  # 1000
```

### 二叉搜索树 (BST)

**定义：** 左子树所有节点小于根，右子树所有节点大于根。

```python
# BST 性质
#     5
#    / \
#   3   7
#  / \ / \
# 1  4 6  8
#
# 中序遍历结果: [1, 3, 4, 5, 6, 7, 8] ← 有序！
```
