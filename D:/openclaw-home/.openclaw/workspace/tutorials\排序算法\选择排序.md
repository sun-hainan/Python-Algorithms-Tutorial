# 选择排序

_每轮选出最小的一个_

## 📖 学习目标

## 第一部分：什么是选择排序？

### 🎯 选择排序的核心思想

**每一轮在未排序部分找到最小的，放到已排序部分的末尾。**

```
第1轮：在所有元素中找到最小的，放到第1位
第2轮：在剩余元素中找到最小的，放到第2位
第3轮：在剩余元素中找到最小的，放到第3位
...
直到所有元素都排好序
```

---

## 第二部分：图解过程

### 例子：将 [64, 25, 12, 22] 排序

```
初始：[64, 25, 12, 22]
      ↓
第1轮：找到最小值 12，与第1位交换
      [12, 25, 64, 22]
      已排序：[12]

第2轮：在 [25, 64, 22] 中找最小值 22，与第2位交换
      [12, 22, 64, 25]
      已排序：[12, 22]

第3轮：在 [64, 25] 中找最小值 25，与第3位交换
      [12, 22, 25, 64]
      已排序：[12, 22, 25]

第4轮：只剩 64，不需要交换
      [12, 22, 25, 64]
      已排序：[12, 22, 25, 64]

排序完成！
```

### 详细图解

```
原数组：[5, 3, 8, 1, 4]

i=0: 在 [5,3,8,1,4] 中找最小 → 1，与 arr[0] 交换
     [1, 3, 8, 5, 4]
     ────
i=1: 在 [3,8,5,4] 中找最小 → 3，无需交换
     [1, 3, 8, 5, 4]
        ───
i=2: 在 [8,5,4] 中找最小 → 4，与 arr[2] 交换
     [1, 3, 4, 5, 8]
           ───
i=3: 在 [5,8] 中找最小 → 5，无需交换
     [1, 3, 4, 5, 8]
              ───

排序完成！
```

## 第三部分：代码逐行解释

```python
def selection_sort(arr):
    """
    选择排序

    参数:
        arr: 待排序列表（原地修改）
    返回值:
        排序后的列表
    """

    n = len(arr)

    # -------- 外层循环：控制轮数 --------
    for i in range(n):

        # -------- 第 i 轮：找到未排序部分的最小值 --------
        min_index = i  # 假设当前位置是最小值

        # -------- 内层循环：在未排序部分找最小值 --------
        for j in range(i + 1, n):
            if arr[j] < arr[min_index]:
                # 发现了更小的，更新最小值索引
                min_index = j

        # -------- 把最小值交换到位置 i --------
        # 如果最小值不在位置 i，才需要交换
        if min_index != i:
            arr[i], arr[min_index] = arr[min_index], arr[i]

    return arr
```

## 第四部分：实际运行代码

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        if min_index != i:
            arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr


# ========================== 测试 ==========================

print("测试1：简单数字排序")
arr1 = [64, 25, 12, 22]
print(f"  排序前: {arr1}")
print(f"  排序后: {selection_sort(arr1)}")
# 输出: [12, 22, 25, 64]

print("\n测试2：选秀节目排名")
scores = [85, 92, 78, 96, 88, 73]
print(f"  初始分数: {scores}")
print(f"  最终排名: {selection_sort(scores)}")
# 输出: [73, 78, 85, 88, 92, 96]

print("\n测试3：已有顺序的数组")
print(f"  排序后: {selection_sort([1, 2, 3])}")
# 输出: [1, 2, 3]
```

## 第五部分：实际应用场景

### 🏆 场景1：选秀节目海选排名

每轮选出当前最高分的选手。

```python
contestants = [85, 92, 78, 96, 88, 73]

# 选择排序思路：
# 第1轮：最高分 96，移到第1位
# 第2轮：次高分 92，移到第2位
# ...

sorted_scores = selection_sort(contestants)
print(f"最终排名: {sorted_scores[::-1]}")  # 倒序输出
# 输出: [96, 92, 88, 85, 78, 73]
```

### 📊 场景2：成绩筛选

找出最低分的几个学生。

```python
# 找出后3名
scores = [85, 92, 78, 96, 88, 73, 65, 80]
sorted_scores = selection_sort(scores)
bottom_3 = sorted_scores[:3]
print(f"后3名: {bottom_3}")  # [65, 73, 78]
```

### 🔧 场景3：简单排序需求

数据量小，不需要高性能排序。

```python
# 小规模数据排序
small_list = [3, 1, 4, 1, 5, 9, 2, 6]
print(f"排序后: {selection_sort(small_list)}")
```

## 第六部分：选择排序 vs 其他排序

### 特点对比

| 特点 | 选择排序 | 冒泡排序 | 插入排序 |
|------|----------|----------|----------|
| 时间复杂度 | O(n²) | O(n²) | O(n²) |
| 交换次数 | O(n) ⭐ | O(n²) | O(n²) |
| 稳定性 | ❌ 不稳定 | ✅ 稳定 | ✅ 稳定 |
| 特点 | 交换少 | 比较多的 | 对有序数据友好 |

### 为什么选择排序不稳定？

```
原始数组：[2₁, 3, 2₂, 1]

第1轮：找到最小值 1，与 arr[0] 交换
      [1, 3, 2₂, 2₁]
      
注意：两个 2 的相对位置变了！
原始：2₁ 在 2₂ 前面
排序后：2₂ 在 2₁ 前面

这就是"不稳定"！
```

### 何时使用选择排序？

✅ **适合使用：**
- 交换成本远高于比较成本
- 数据量小
- 不要求稳定性

❌ **不适合使用：**
- 大规模数据
- 需要稳定排序

## 第七部分：名词解释

### 不稳定排序

**定义：** 排序后，相等元素的相对位置可能改变。

```python
# 选择排序不稳定示例
arr = [2₁, 3, 2₂, 1]
# 排序后可能变成：[1, 3, 2₂, 2₁]
# 2₁ 和 2₂ 的位置互换了
```

### 交换次数

**定义：** 元素被移动的次数。

```python
# 选择排序最多需要 n-1 次交换
for i in range(n):
    # 最多一次交换
    arr[i], arr[min_index] = arr[min_index], arr[i]
```

### 内层循环 vs 外层循环

```python
for i in range(n):           # 外层：控制轮数，n-1 轮
    min_index = i
    for j in range(i+1, n):  # 内层：在未排序部分找最小
        ...
```

---

## ✅ 小结

1. **选择排序**每轮找未排序部分的最小值
2. 把最小值交换到已排序部分末尾
3. **交换次数少**（最多 n-1 次）
4. **不稳定**排序
5. 时间复杂度 O(n²)，但简单直观
