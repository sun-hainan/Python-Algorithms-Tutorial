# 最长递增子序列

_Longest Increasing Subsequence_

## 第一部分：什么是 LIS？

### 📝 定义

**递增子序列：** 子序列中的元素单调递增。

**最长递增子序列：** 长度最长的递增子序列。

### 例子

```
原序列: [10, 9, 2, 5, 3, 7, 101, 18]

递增子序列:
- [10, 101]
- [2, 5, 7, 101]
- [2, 3, 7, 101]
- ...

最长递增子序列: [2, 5, 7, 101] 或 [2, 3, 7, 101]
长度: 4
```

## 第二部分：两种解法

### 解法1：动态规划 O(n²)

```python
def lis_dp(arr):
    """
    LIS - 动态规划解法
    时间复杂度: O(n²)
    """
    if not arr:
        return 0

    n = len(arr)
    dp = [1] * n

    for i in range(1, n):
        for j in range(i):
            if arr[j] < arr[i]:
                dp[i] = max(dp[i], dp[j] + 1)

    return max(dp)
```

### 解法2：二分优化 O(n log n)

```python
import bisect

def lis_binary(arr):
    """
    LIS - 二分查找优化
    时间复杂度: O(n log n)
    """
    if not arr:
        return 0

    tail = []

    for num in arr:
        pos = bisect.bisect_left(tail, num)
        if pos == len(tail):
            tail.append(num)
        else:
            tail[pos] = num

    return len(tail)
```

## 第三部分：实际运行代码

```python
arr = [10, 9, 2, 5, 3, 7, 101, 18]

print(f"数组: {arr}")
print(f"LIS 长度 (DP): {lis_dp(arr)}")
print(f"LIS 长度 (二分): {lis_binary(arr)}")
```

## 第四部分：应用场景

### 📈 股票最长上涨期

```python
prices = [100, 180, 260, 310, 40, 535, 695]
print(f"最长连续上涨天数: {lis_binary(prices)}")
```

### 🎤 合唱队问题

```python
# 选最多的人站成一排，身高递增
heights = [170, 165, 180, 160, 175, 172]
print(f"最多选 {lis_dp(heights)} 人")
```

## 第五部分：算法对比

| 解法 | 时间复杂度 | 空间复杂度 |
|------|-----------|------------|
| DP | O(n²) | O(n) |
| 二分 | O(n log n) ⭐ | O(n) |

## ✅ 小结

1. **LIS** 找最长递增子序列
2. 两种解法：O(n²) DP 和 O(n log n) 二分
3. 应用：股票分析、合唱队问题
