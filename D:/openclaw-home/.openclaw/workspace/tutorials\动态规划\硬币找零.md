## 第一部分：问题定义

### 📝 问题描述

给定几种面值的硬币，每种无限多：
- 硬币面值：c1, c2, c3, ..., ck
- 目标金额：n
- 求：凑出 n 所需的最少硬币数

### 例子

```
硬币面值：[1, 5, 10, 25]
目标金额：63

贪心解：25 + 25 + 10 + 1 + 1 + 1 = 6枚
动态规划解：25 + 25 + 10 + 1 + 1 + 1 = 6枚
```

---

## 第二部分：解法对比

### 解法1：贪心法

```python
def coin_change_greedy(amount, coins=[1, 5, 10, 25]):
    """
    贪心解法：每次选面值最大的

    特点：简单快速，但不一定最优
    """
    coins = sorted(coins, reverse=True)
    count = 0
    remaining = amount

    for coin in coins:
        num = remaining // coin
        count += num
        remaining -= num * coin

    return count if remaining == 0 else -1


# 测试
print(coin_change_greedy(63))  # 6
```

### 解法2：动态规划

```python
def coin_change_dp(amount, coins=[1, 5, 10, 25]):
    """
    动态规划解法

    dp[i] = 凑出金额 i 所需的最少硬币数
    """
    # 初始化：dp[0] = 0，其他为无穷大
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0

    # 遍历所有金额
    for i in range(1, amount + 1):
        # 遍历所有硬币
        for coin in coins:
            if i >= coin:
                dp[i] = min(dp[i], dp[i - coin] + 1)

    return dp[amount] if dp[amount] != float('inf') else -1


# 测试
print(coin_change_dp(63))  # 6
```

### 解法3：找出具体方案

```python
def coin_change_solution(amount, coins=[1, 5, 10, 25]):
    """
    动态规划 + 路径记录
    """
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    parent = [-1] * (amount + 1)  # 记录每个金额的前一个

    for i in range(1, amount + 1):
        for coin in coins:
            if i >= coin and dp[i - coin] + 1 < dp[i]:
                dp[i] = dp[i - coin] + 1
                parent[i] = coin

    if dp[amount] == float('inf'):
        return []

    # 回溯找出方案
    solution = []
    current = amount
    while current > 0:
        solution.append(parent[current])
        current -= parent[current]

    return solution


# 测试
print(coin_change_solution(63))
# 输出：[25, 25, 10, 1, 1, 1]
```

## 第三部分：为什么贪心不一定最优？

### ⚠️ 贪心失效的例子

```
硬币面值：[1, 3, 4]
目标金额：6

贪心解：4 + 1 + 1 = 3枚
最优解：3 + 3 = 2枚

结论：贪心选 4，但应该选两个 3
```

```python
# 贪心
def greedy_wrong():
    coins = [1, 3, 4]
    amount = 6
    coins = sorted(coins, reverse=True)
    count = 0
    remaining = amount
    for coin in coins:
        num = remaining // coin
        count += num
        remaining -= num * coin
    return count

print(greedy_wrong())  # 3 (不是最优)

# 动态规划
print(coin_change_dp(6, [1, 3, 4]))  # 2 (最优)
```

## 第四部分：完全背包 vs 0-1背包

### 完全背包（硬币找零）

```python
def coin_change_complete(amount, coins):
    """
    完全背包：每种硬币无限个

    dp[i] = 凑出 i 所需的最少硬币数
    """
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0

    for coin in coins:  # 先遍历硬币
        for i in range(coin, amount + 1):  # 再遍历金额
            dp[i] = min(dp[i], dp[i - coin] + 1)

    return dp[amount]
```

### 0-1 背包（每种物品只能选一次）

```python
def knapsack_01(W, weights, values, n):
    """
    0-1背包

    dp[i][w] = 前 i 个物品，背包容量 w 的最大价值
    """
    dp = [[0] * (W + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for w in range(W + 1):
            if weights[i-1] <= w:
                dp[i][w] = max(dp[i-1][w], dp[i-1][w-weights[i-1]] + values[i-1])
            else:
                dp[i][w] = dp[i-1][w]

    return dp[n][W]
```

## 第五部分：实际应用场景

### 💰 场景1：自动售货机找零

```python
def vending_machine(price, paid, coins=[1, 5, 10, 50, 100]):
    """
    自动售货机找零
    """
    change = paid - price
    if change < 0:
        return "金额不足"

    result = coin_change_solution(change, coins)
    return f"找零：{result}" if result else "无法找零"


print(vending_machine(63, 100))  # 找零：[25, 10, 1, 1]
```

### 🎁 场景2：红包金额分配

```python
def distribute_red_packet(total, count, denominations=[1, 5, 10, 50, 100]):
    """
    红包分配：把总金额分成 count 份
    """
    if total < count:
        return []

    # 每人至少1元
    remaining = total - count
    distribution = [1] * count

    # 贪心分配
    coins = sorted(denominations, reverse=True)
    for coin in coins:
        num = remaining // coin
        if num > 0:
            # 优先给金额大的
            for i in range(count):
                if num > 0 and distribution[i] < 100:
                    extra = min(num, 3)
                    distribution[i] += coin * extra
                    num -= extra
                if num == 0:
                    break

    return distribution
```

## 第六部分：名词解释

### 完全背包

**定义：** 每种物品（硬币）可以无限使用。

### 0-1 背包

**定义：** 每种物品只能选 0 个或 1 个。

### 状态转移方程

```python
dp[i] = min(dp[i], dp[i - coin] + 1)
# 凑出金额 i = min(原来需要的硬币数, 用一枚 coin 后的硬币数 + 1)
```

## ✅ 小结

1. **硬币找零**是动态规划的经典入门问题
2. **贪心不一定最优**（如面值 [1,3,4]）
3. **动态规划**保证找到最优解
4. 完全背包 vs 0-1 背包：遍历顺序不同
