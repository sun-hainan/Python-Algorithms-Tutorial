## 📖 核心概念

### 动态规划三要素

| 要素 | 说明 | 示例 |
|------|------|------|
| 最优子结构 | 大问题的最优解包含小问题的最优解 | 背包：选/不选 |
| 重叠子问题 | 子问题会被多次计算 | Fibonacci |
| 状态转移 | 大问题和小问题的关系 | dp[n] = dp[n-1] + dp[n-2] |
