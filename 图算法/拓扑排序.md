## 第一部分：什么是拓扑排序？

### 📝 问题定义

**拓扑排序：** 对有向无环图(DAG)进行线性排序，使得所有有向边(u,v)中，u都在v之前。

## 第二部分：算法实现

### Kahn 算法（BFS 版本）

```python
from collections import deque

def topological_sort(graph):
    """
    拓扑排序 - Kahn 算法

    参数:
        graph: 邻接表 {顶点: [邻接顶点]}
    返回值:
        拓扑排序结果
    """

    # -------- 计算入度 --------
    in_degree = {v: 0 for v in graph}
    for v in graph:
        for neighbor in graph[v]:
            in_degree[neighbor] += 1

    # -------- 入度为0的顶点入队 --------
    queue = deque([v for v in graph if in_degree[v] == 0])
    result = []

    # -------- BFS --------
    while queue:
        vertex = queue.popleft()
        result.append(vertex)

        for neighbor in graph[vertex]:
            in_degree[neighbor] -= 1
            if in_degree[neighbor] == 0:
                queue.append(neighbor)

    # -------- 检查是否有环 --------
    if len(result) != len(graph):
        return []  # 有环，无法拓扑排序

    return result
```

### DFS 版本

```python
def topological_sort_dfs(graph):
    """
    拓扑排序 - DFS 版本
    """
    visited = set()
    result = []
    on_stack = set()

    def dfs(vertex):
        if vertex in on_stack:
            return False  # 有环
        if vertex in visited:
            return True
        visited.add(vertex)
        on_stack.add(vertex)
        for neighbor in graph.get(vertex, []):
            if not dfs(neighbor):
                return False
        on_stack.remove(vertex)
        result.append(vertex)  # 后序遍历
        return True

    for v in graph:
        if v not in visited:
            if not dfs(v):
                return []  # 有环

    return result[::-1]
```

| 方法 | 思想 | 优点 |

## 第六部分：名词解释

### 有向无环图 (DAG)

**定义：** 没有环的有向图。

### 入度

**定义：** 指向该顶点的边的数量。

## ✅ 小结

1. 拓扑排序用于**有依赖关系的任务排序**
2. 适用于**有向无环图(DAG)**
3. 两种实现：**Kahn算法(BFS)** 和 **DFS算法**
4. 如果排序失败，说明**有环**
