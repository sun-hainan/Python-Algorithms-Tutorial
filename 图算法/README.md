# 图算法

_处理关系网络的算法_

---

## 📋 算法列表

| 算法 | 时间复杂度 | 用途 |
|------|-----------|------|
| BFS 广度优先搜索 | O(V+E) | 最短路径、层级遍历 |
| DFS 深度优先搜索 | O(V+E) | 连通性、拓扑排序 |
| Dijkstra 最短路径 | O((V+E) log V) | 单源最短路径（非负权）|
| Floyd-Warshall | O(V³) | 全源最短路径 |
| 拓扑排序 | O(V+E) | 依赖排序 |

## 🔑 核心概念

### 图的表示

```python
# 邻接表
graph = {
    'A': ['B', 'C'],
    'B': ['C', 'D'],
    'C': ['D'],
    'D': []
}

# 邻接矩阵
#    A  B  C  D
# A  0  1  1  0
# B  0  0  1  1
# C  0  0  0  1
# D  0  0  0  0
```

### BFS vs DFS

```
BFS（队列）：一层一层往下走
Level 0: A
Level 1: B, C
Level 2: D, E
→ 适合找最短路径

DFS（栈/递归）：一条路走到黑
A → B → D → 返回 → C → E
→ 适合遍历、连通性检查
```
